## 第一性原理（First Principles）

PR 理论不能从循环定义出发，必须有一个不可再约的起点。以下是第一性原理：

### 原理一：物理存在公理（Axiom of Physical Existence）

**陈述**：存在 = 能量密度不为零的物理域。

**操作化定义**：
- 给定一个空间区域 R，测量其能量密度 ρ(R)
- 如果 ρ(R) > 0，则 R 是存在域
- 如果 ρ(R) = 0，则 R 是真空域（非存在域的物理实例）
- 真空不是"无"，而是能量基态。ρ(vacuum) = 0 是物理态，不是非物理态

**与标准物理的对应**：真空能量密度 ~ 10⁻⁹ J/m³（宇宙学常数相关）

---

### 原理二：非存在补集公理（Axiom of Non-Existence as Complement）

**陈述**：非存在 = 存在域的物理补集，不是虚无。

**操作化定义**：
- 给定存在域 E（ρ(E) > 0），其补集 ¬E = 全空间中 E 以外的区域
- ¬E 不是"什么都没有"，而是"与 E 的物理约束不兼容的态集合"
- ¬E 的物理结构由因果约束决定：E 的因果律限制了 ¬E 中可能发生的物理过程

**物理实例**：
- 黑洞事件视界内部（对外部观察者而言）= 非存在域（信息不可达）
- 宇宙视界之外 = 非存在域（因果不连通）
- 这些不是虚无，而是因果约束下的物理补集

---

### 原理三：无自我引用公理（Axiom of No Self-Reference）

**陈述**：任何定义中，被定义项不能出现在定义项中。

**操作化**：
- 定义 D(T) = 描述术语 T 的陈述集合
- 如果 T ∈ D(T)，则 D(T) 是循环定义，无效
- 所有定义必须引用外部可测量的概念

**应用**：量子态论、语言流形论、双态学论的原定义违反此公理，已全部修正（见 v3.1）

---

### 原理四：信息即物理公理（Axiom of Information as Physical）

**陈述**：信息处理是物理过程的一种组织模式，不是物理之外的实在。

**操作化**：
- 信息 = 物理系统的状态差异（可区分的状态数）
- 信息处理 = 物理系统状态之间的转换规则
- 语言 = 物理系统（神经、计算机）的信息处理结构的数学表示
- 没有"非物理的信息"，没有"非物理的语言"

**与标准物理的对应**：Landauer 原理（信息擦除需要能量）、黑洞信息悖论

## 宇宙常数（Cosmic Constants）

PR 理论需要一个基准常数锚定整个理论。候选常数：

### 常数一：存在阈值 Λ_E（Existence Threshold）

**定义**：Λ_E = 10⁻¹²⁰（无量纲，约等于宇宙学常数/普朗克能量密度）

**作用**：
- 存在域判定标准：ρ(R) > ρ_Planck × Λ_E = 存在
- ρ(R) ≤ ρ_Planck × Λ_E = 真空/非存在域
- 这个阈值不是任意的，而是宇宙学观测值

**可解释性**：
- Λ_E 为什么是这个值？= 宇宙学微调问题
- PR 理论的回答：Λ_E 不是基本常数，而是存在/非存在相变的临界值
- 在相变点，存在域和非存在域的能量密度相等，系统处于不稳定态

---

### 常数二：双态耦合常数 g_DE（Dual-state Coupling）

**定义**：g_DE = 描述存在域 E 和非存在域 ¬E 之间耦合强度的无量纲常数

**作用**：
- g_DE = 0：E 和 ¬E 完全解耦（无信息交换，无因果联系）
- g_DE = 1：E 和 ¬E 完全耦合（统一为一态）
- 0 < g_DE < 1：部分耦合（双态学论研究的区域）

**物理实例**：
- 量子纠缠 = 两个粒子之间的 g_DE ≈ 1（高度耦合）
- 黑洞蒸发 = 黑洞内（E）与外部（¬E）的 g_DE 从 0 逐渐增加到 1（信息泄漏）
- 宇宙膨胀 = 可观测宇宙（E）与视界外（¬E）的 g_DE ≈ 0（因果解耦）

**可解释性**：
- g_DE 不是基本常数，而是特定物理系统的涌现属性
- 不同的物理系统有不同的 g_DE 值
- PR 理论的目标是建立一个计算 g_DE 的框架

---

### 常数三：形式化完备常数 C_F（Formalization Completeness）

**定义**：C_F = PR 理论中已形式化陈述 / 总陈述数

**作用**：
- C_F = 0：完全未形式化（纯哲学）
- C_F = 1：完全形式化（可编译为 Lean）
- 当前 PR 理论的 C_F ≈ 0.1（大部分陈述尚未形式化）

**目标**：
- v3.2 目标：C_F ≥ 0.3（三基元代数、层级结构、几何图形式化）
- v4.0 目标：C_F ≥ 0.7（所有子理论的形式化表述）
- v5.0 目标：C_F = 1（完整 Lean 实现）

## 修正后的核心公理（v3.2 完整版）

1. **物理存在公理**：存在 = 能量密度 > 0 的物理域。不存在语言之外的实在。
2. **非存在补集公理**：非存在 = 存在域的物理补集。不是虚无。补集有物理结构（因果约束）。
3. **无自我引用公理**：定义不能引用自身。循环定义 = 诡辩，无效。
4. **信息即物理公理**：信息处理是物理过程的组织模式。没有非物理的信息。
5. **操作化公理**：概念必须还原为可测量的操作。不可操作化的概念 = 未定义。

## 修正后的何道（v3.2）

**硬回答**：
- "道" = 物理态的实在结构，不是语言的构造
- 语言可以描述道，但不能创造道
- 道在语言之外
- 何道 = 问"道是什么"这个问题本身是一个物理操作（神经放电），但问题的答案指向物理实在，不是语言构造

**不存在循环**：
- 问"道是什么" ≠ 道本身
- 问题（语言操作）≠ 答案（物理实在）
- 没有自我指涉，没有诡辩
