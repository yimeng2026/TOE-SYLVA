## 二、常数之间的精确理论关系

### 2.1 电磁学常数关系网

```
α = e² / (4π ε₀ ℏ c)            [精细结构常数的经典定义]
  = (μ₀ c e²) / (2 h)           [用真空磁导率表示]
  = e² c μ₀ / (2 h)

R_K = h / e² = μ₀ c / (2α)      [冯·克利青常数与α的关系]

K_J = 2e / h = 1/Φ₀            [约瑟夫森常数与磁通量子]

Φ₀ = h / (2e)                   [磁通量子]

Z₀ = √(μ₀/ε₀) = μ₀ c = 376.730 313 461... Ω  [真空阻抗]

α = Z₀ e² / (2 h)               [用真空阻抗表示]
```

**真空电磁常数关系**:
```
c = 1/√(ε₀ μ₀)                  [光速与真空电磁常数]
ε₀ = 1/(μ₀ c²)                  [精确导出 (2019 SI后 μ₀变为待测定)]
μ₀ = 2α h / (c e²)              [μ₀不再是精确值]
```

设定四个常数为1：G = ℏ = c = k_B = 1

**意义**: Planck尺度 (~10⁻³⁵ m) 是量子引力效应变得显著的尺度，是弦理论和圈量子引力的特征尺度。
