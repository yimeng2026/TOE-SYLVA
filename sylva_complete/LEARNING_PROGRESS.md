## 📝 学习笔记

### 常用策略速查表

| 策略 | 用途 | 示例 |
|------|------|------|
| rfl | 自反性 | `rfl` |
| simp | 简化 | `simp [h1, h2]` |
| rw | 重写 | `rw [h]` |
| linarith | 线性不等式 | `linarith` |
| nlinarith | 非线性不等式 | `nlinarith` |
| ring | 环代数 | `ring` |
| calc | 链式推导 | `calc a = b := ...` |
| have | 辅助引理 | `have h : P := ...` |
| intro | 引入假设 | `intro h` |
| apply | 应用定理 | `apply theorem_name` |

### 常见错误与解决

| 错误 | 原因 | 解决 |
|------|------|------|
| type mismatch | 类型不匹配 | 检查变量类型，可能需要转换 |
| unknown identifier | 未定义标识符 | 检查拼写，确保已导入 |
| tactic failed | 策略失败 | 尝试其他策略，或分解证明 |
