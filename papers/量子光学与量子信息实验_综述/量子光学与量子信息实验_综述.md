# 量子光学与量子信息实验综述

## 摘要

量子光学与量子信息实验作为当代物理学最前沿的研究领域之一，在过去二十年间取得了令人瞩目的进展。本文系统综述了该领域的核心实验技术、关键科学突破以及未来发展挑战。从单光子源与量子纠缠的制备，到量子密钥分发、量子计算原型机与量子网络的构建，实验量子光学已逐步从基础研究走向技术应用。本文梳理了量子态操控、量子测量、量子存储与量子中继等关键技术节点的实验进展，分析了量子优势演示、量子纠错初步实现以及多节点量子网络互联等重要里程碑。同时，本文引入SYLVA本体论框架，建立了量子光学与量子信息实验分支与其他SYLVA学科模块的交叉联系表，揭示了该领域在TOE-SYLVA统一知识体系中的枢纽地位。最后，本文讨论了可扩展性、集成化与实用化等核心挑战，并展望了量子互联网与分布式量子计算的未来图景。

**关键词**：量子光学；量子信息；量子纠缠；量子计算；量子通信；量子网络；量子精密测量；SYLVA本体论

## 一、引言

量子光学与量子信息实验是连接量子力学基础理论与量子技术实际应用的桥梁。自爱因斯坦（Albert Einstein）、波多尔斯基（Boris Podolsky）和罗森（Nathan Rosen）于一九三五年提出EPR悖论以来，量子纠缠这一非经典关联便持续激发着物理学界的深刻思考。薛定谔（Erwin Schrödinger）同年将其命名为Verschränkung（纠缠），指出其是量子力学的本质特征。直至贝尔（John Stewart Bell）于一九六四年提出可实验检验的贝尔不等式，以及克劳泽（John Clauser）、霍恩（Michael Horne）、希莫尼（Abner Shimony）和霍尔特（Richard Holt）于一九六九年提出CHSH不等式，量子纠缠才从哲学思辨走向实验可证伪的科学命题。一九八二年间，阿斯佩（Alain Aspect）等人完成了里程碑式的贝尔不等式违反实验，为量子信息科学的诞生奠定了实验基础。

进入二十一世纪，随着单光子探测、量子存储与集成光子学等核心技术的突破，量子光学与量子信息实验领域迎来了爆发式增长。量子密钥分发（QKD）已实现商业部署；量子计算原型机展现出量子优势；量子中继与量子网络的原型系统正在全球构建。本文旨在综述该领域的实验进展，建立与TOE-SYLVA知识体系的本体论联系，并探讨其发展方向。

## 二、量子光学的核心实验基础

### 2.1 单光子源与光子态制备

单光子源是量子光学实验的基本构件。理想的单光子源应以确定性方式每次发射且仅发射一个光子，并满足高纯度、高不可区分性与高效率的要求。当前实验实现主要分为以下几类：

（一）**参数下转换源（SPDC）**：利用非线性晶体（如BBO、PPKTP）中的二阶非线性过程，将泵浦光子劈裂为一对信号光子和闲频光子。该过程满足能量守恒和动量守恒（相位匹配条件）：

$$\omega_p = \omega_s + \omega_i, \quad \vec{k}_p = \vec{k}_s + \vec{k}_i$$

SPDC源具有纠缠光子对产生能力，但属于概率性光源，存在多光子污染问题。通过周期性极化晶体和波导结构增强非线性效率，是当前提升SPDC源性能的重要方向。

（二）**量子点单光子源（QD）**：基于III-V族半导体异质结结构（如InAs/GaAs），利用三维量子限制效应实现单激子的辐射复合。量子点源具有接近理想的发射特性：单光子发射纯度可达$g^{(2)}(0) < 0.01$，光子不可区分性经光谱滤波和电场调控后可超过$0.98$。在液氦温度下，量子点单光子源已实现与光纤网络的高效耦合，是构建量子中继节点的理想选择。

（三）**色心单光子源**：以金刚石氮空位（NV）中心、硅空位（SiV）中心为代表，利用固态晶体缺陷中的单电子自旋作为发射中心。色心源具有室温操作、长相干时间、可光学读出与操控等优势，在量子网络和量子传感领域展现出独特价值。近年来，利用金刚石纳米结构增强收集率（Purcell效应），NV中心的单光子收集率已提升至接近$100\%$。

### 2.2 量子纠缠的制备与操控

量子纠缠是量子信息资源的核心载体。在光学系统中，偏振纠缠、时间-能量纠缠、轨道角动量（OAM）纠缠和路径纠缠等多种自由度已被实验实现。偏振纠缠是最常用的纠缠形式，典型的贝尔态可表示为：

$$|\Psi^-\rangle = \frac{1}{\sqrt{2}}(|H\rangle_1|V\rangle_2 - |V\rangle_1|H\rangle_2)$$

$$|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|H\rangle_1|H\rangle_2 + |V\rangle_1|V\rangle_2)$$

其中$|H\rangle$和$|V\rangle$分别表示水平偏振和垂直偏振光子。通过Sagnac干涉仪或双路SPDC方案，偏振纠缠源已实现超过$99\%$的纠缠保真度和每秒百万量级的光子对产生率。高维纠缠（qudit）利用光子的轨道角动量或频率自由度扩展希尔伯特空间维度。二零二零年，潘建伟团队实验实现了$15$维OAM纠缠的量子密集编码，信道容量超过二维系统极限。

### 2.3 量子态测量与层析重建

量子态测量是提取量子信息的关键环节。对于量子光学系统，量子态层析（QST）通过在不同测量基下重复投影测量，重建量子态密度矩阵$\rho$。对于单光子偏振态，至少需要$4$个线性无关的投影测量（Stokes参数）实现完全态重构；对于双光子纠缠态，需在$16$个组合基下测量coincidence count。实验上，量子态层析已能实现优于$0.999$的保真度估计精度。

## 三、量子信息处理的关键实验进展

### 3.1 量子计算与量子模拟

量子计算旨在利用量子叠加和量子纠缠实现超越经典计算的信息处理能力。实验量子计算已从原理验证走向中等规模含噪声量子（NISQ）时代。

**光量子计算**以线性光学量子计算（LOQC）为代表，利用单光子、分束器和相位调制器实现量子逻辑门。KLM方案（Knill-Laflamme-Milburn）证明，通过量子隐形传态和辅助光子，可以以概率性方式实现通用量子计算。后续发展的簇态（Cluster State）单向量子计算，仅需制备大规模纠缠态后执行自适应单比特测量。实验上，已演示了$12$光子簇态上的玻色采样和$18$光子图态的量子纠错编码。

**超导量子计算**基于约瑟夫森结构建的人工原子具有GHz级别的能级间距，可通过微波脉冲实现快速量子门操作。Google的Sycamore处理器集成$53$个超导量子比特，在$200$秒内完成了特定随机线路采样任务，声称实现了量子优势。IBM的Eagle处理器已扩展至$127$个量子比特，并展示了动态量子电路中的纠错编码。超导量子比特的相干时间$T_1$和$T_2$已分别达到数百微秒量级，单比特和双比特门保真度超过$99.5\%$和$99\%$。

**离子阱量子计算**利用射频Paul阱囚禁单个离子，其内态作为量子比特，通过激光实现量子门操作。离子阱系统具有长相干时间（秒至分钟量级）、全连通性和高保真度（单比特门$99.9999\%$，双比特门$99.9\%$）的优势。Quantinuum的H2系统已演示了$32$个全连接量子比特的纠缠态制备。Molmer-Sorensen门是离子阱系统中实现双比特纠缠的标准方案：

$$U_{MS} = \exp\left(-i\frac{\pi}{4}\sigma_x^{(1)}\sigma_x^{(2)}\right)$$

**中性原子量子计算**利用光镊阵列捕获和操控单个中性原子，通过Rydberg阻塞效应实现两原子间的可控相位门。中性原子系统具有规模可扩展性（已实现$1000+$原子阵列）和快速重排能力。哈佛-麻省理工超冷原子中心演示了$48$个逻辑量子比特的量子纠错码，并实现了$280$个物理比特的量子模拟。

量子模拟方面，利用超冷原子在光学晶格中已实现反铁磁相关联的观测和高温超导机理的模拟研究。费米子-哈伯德模型的量子模拟已取得突破性进展，其哈密顿量为$H = -t\sum_{\langle i,j \rangle, \sigma}(c_{i,\sigma}^\dagger c_{j,\sigma} + h.c.) + U\sum_i n_{i,\uparrow}n_{i,\downarrow}$。

### 3.2 量子通信与量子密码

量子通信利用量子态的不可克隆定理和测量坍缩特性实现信息的安全传输。BB84协议由Bennett和Brassard于一九八四年提出，利用两组非正交基编码量子比特：

$$|0\rangle, |1\rangle \quad \text{和} \quad |+\rangle = \frac{|0\rangle + |1\rangle}{\sqrt{2}}, |-\rangle = \frac{|0\rangle - |1\rangle}{\sqrt{2}}$$

窃听者不可避免地引入误码率，可被合法通信双方通过误码率监测发现。信息论安全性证明表明，当误码率低于约$11\%$时，可通过隐私放大提取安全密钥。

实验QKD系统已在多个物理平台上实现。**光纤QKD**基于弱相干光源或诱骗态方案，传输距离已突破$500$km（通过双场QKD, TF-QKD）。中国"京沪干线"量子通信骨干网全长$2000$余公里，实现了城际量子密钥分发。**自由空间QKD**利用卫星作为中继节点，突破了光纤的地理限制。中国"墨子号"量子科学实验卫星实现了星地$1200$km级别的量子密钥分发，并与奥地利地面站完成了洲际量子通信。**芯片集成QKD**利用硅光子学平台实现片上集成，是消费级量子密码设备的关键路径。

量子隐形传态利用纠缠对和经典通信通道实现量子态的远距离传输。对于未知量子态$|\psi\rangle = \alpha|0\rangle + \beta|1\rangle$，其隐形传态过程可表述为$|\psi\rangle_A|\Phi^+\rangle_{BC} = \frac{1}{2}\sum_{i=0}^{3}|\Phi_i\rangle_{AB}\sigma_i|\psi\rangle_C$，其中$|\Phi_i\rangle$为四个贝尔态，$\sigma_i$为相应的泡利修正操作。实验上，量子隐形传态已在光纤和自由空间链路中实现了超过$100$km的传输距离，保真度超过$90\%$。

### 3.3 量子纠错与容错量子计算

量子纠错是实现大规模通用量子计算的必由之路。表面码（Surface Code）是当前最具前景的量子纠错方案，仅需最近邻双比特门和单比特测量即可实现。在$d\times d$的方形格点上，逻辑错误率随物理错误率$p$呈指数抑制：$p_L \sim \left(\frac{p}{p_{th}}\right)^{(d+1)/2}$，其中$p_{th}$为阈值错误率（约$0.5\%$至$1\%$）。Google于二零二四年实验演示了$49$个物理量子比特构成的距离-5表面码，逻辑错误率低于距离-3码，首次实现了量子纠错的扩展优势。色心平台的量子纠错也取得重要进展，利用金刚石NV中心的电子自旋和邻近核自旋，已实现了重复码和$5$量子比特Steane码的编码与纠错演示。

## 四、量子存储与量子网络实验

### 4.1 量子存储技术

量子存储是实现量子中继和量子网络的核心技术，要求对量子态进行可逆的写入、保持和读出操作，并保持高保真度。

**冷原子系综存储**利用电磁诱导透明（EIT）效应实现光子与集体原子激发的可逆转换。在EIT介质中，群速度被大幅降低，光子波包被压缩进入原子系综：$v_g = \frac{c}{1 + \frac{g^2N}{|\Omega_c|^2}} \ll c$。当控制光绝热关闭时，光子信息被映射为原子自旋波。通过光泵浦技术，冷原子系综量子存储的效率已超过$50\%$，保真度超过$95\%$，存储时间达到毫秒至秒量级。

**固态量子存储**基于稀土掺杂晶体（如Pr$^{3+}$:Y$_2$SiO$_5$, Eu$^{3+}$:Y$_2$SiO$_5$）或色心系统。稀土离子具有超窄的光学跃迁线宽（kHz量级）和超长相干时间（数小时），是实现长寿命量子存储的优异平台。利用零一阶塞曼（ZEFOZ）技术，已实现了$6$小时的量子存储相干时间，创造了固态量子存储的世界纪录。

### 4.2 量子中继与量子网络

量子中继通过纠缠交换（Entanglement Swapping）和纠缠纯化（Entanglement Purification）克服光子传输损耗和退相干，实现远距离量子通信。纠缠交换利用两个独立纠缠对之间的贝尔态测量实现远距离纠缠的桥接：$|\Phi^+\rangle_{AB} \otimes |\Phi^+\rangle_{CD} \xrightarrow{BSM(B,C)} |\Phi^+\rangle_{AD}$。量子网络将量子节点通过量子信道互联，实现量子信息的分布式处理和共享。量子网络的架构通常分为三层：量子物理层（物理连接和纠缠分发）、量子链路层（纠缠纯化和拓扑管理）和量子网络层（端到端量子通信和量子任务编排）。

实验进展方面，Delft-QuTech于二零一五年实现了基于NV中心的第一个量子网络节点；中国科学技术大学构建了世界首个量子通信广域网，通过"墨子号"卫星和地面光纤链路实现了多城市间的量子密钥分发；美国DOE资助的量子网络测试床正在建设基于光纤和自由空间的量子网络原型。

## 五、量子精密测量与量子传感

量子精密测量利用量子叠加和量子纠缠突破经典测量极限，实现引力波探测、生物磁场成像、惯性导航等应用的性能跃升。

### 5.1 量子计量学与标准量子极限

对于$N$个独立量子探针的相位测量，经典策略的测量精度受限于散粒噪声：$\Delta\phi \geq \frac{1}{\sqrt{N}}$，这被称为标准量子极限（SQL）。利用量子纠缠，特别是NOON态，可实现海森堡极限：$\Delta\phi \geq \frac{1}{N}$。海森堡极限比标准量子极限提高了$\sqrt{N}$倍，代表了量子力学允许的最佳精度。实验上，利用双模压缩态和纠缠光子对，已在引力波探测和光学相位测量中实现了SQL的突破。

### 5.2 量子传感实验平台

**原子钟与光晶格钟**：基于窄线宽光学跃迁的锶原子光晶格钟已实现$10^{-18}$量级的频率不确定度，比当前秒定义（铯原子微波钟）高出两个数量级。光晶格钟利用Lamb-Dicke区域的束缚抑制多普勒效应，囚禁频率为$\nu = \frac{1}{2\pi}\sqrt{\frac{2V_0}{m\lambda_L^2}}$，其中$V_0$为晶格势阱深度，$\lambda_L$为激光波长。光晶格钟在相对论大地测量、深空导航和基本常数检验中具有重要应用。

**原子干涉仪重力仪**：利用物质波干涉（如Mach-Zehnder干涉）实现重力加速度$g$的精密测量。原子干涉仪的相位为$\Delta\Phi = \vec{k}_{eff} \cdot \vec{g} T^2$，其中$\vec{k}_{eff}$为有效波矢，$T$为干涉脉冲间隔。冷原子干涉重力仪的灵敏度已达到$10^{-9} g/\sqrt{Hz}$，在资源勘探和惯性导航中展现出应用前景。

**量子磁力计**：基于NV色心的光探测磁共振（ODMR）技术，可实现室温下皮特斯拉（pT）灵敏度的磁场测量。NV色心磁力计具有亚微米空间分辨率和毫秒时间分辨率，在生物磁成像和材料科学中应用广泛。此外，基于$^{87}$Rb原子自旋的SERF磁力计已实现亚飞特斯拉（fT）灵敏度，成为最灵敏的磁场探测器之一。

## 七、挑战与展望

尽管量子光学与量子信息实验取得了显著进展，但在迈向实用化量子技术的过程中仍面临多重挑战：

**（一）可扩展性瓶颈**：当前量子系统的规模和集成度仍远未达到实用化量子计算所需的上百万物理量子比特。光子损耗、量子比特串扰和控制系统复杂度是主要障碍。片上集成光子学和模块化量子架构是潜在解决路径。

**（二）噪声与退相干**：环境噪声和热涨落导致的量子退相干限制了量子操作的保真度和量子存储的寿命。虽然量子纠错理论已较为成熟，但物理错误率仍需降低一个数量级才能达到容错阈值。材料科学进步和动态解耦技术是重要研究方向。

**（三）标准化与互操作性**：不同物理平台之间的量子态转换和互操作性尚未成熟。量子网络需要统一的协议栈和接口标准，类似经典互联网的分层架构正在制定中，但各层之间的协议映射仍需大量实验探索。

**（四）成本与工程化**：量子光学实验设备成本高昂、体积庞大，限制了技术推广。室温固态量子系统和紧凑型光纤集成方案是降低成本的重要方向。

展望未来，量子光学与量子信息实验将沿着以下方向纵深发展：**量子互联网**构建全球范围的量子通信基础设施，实现量子计算机的分布式互联和量子资源共享；**室温量子计算**基于色心、激子极化激元等室温量子系统，摆脱低温设备束缚，实现桌面级量子计算；**量子-经典混合计算**将量子处理器作为经典计算的专用加速器，在量子化学模拟、优化问题和机器学习等领域发挥量子优势；**量子生物传感**利用量子传感器的高灵敏度实现单分子磁共振成像和神经信号检测，开启生命科学的量子时代。

## 八、结论

量子光学与量子信息实验已从验证量子力学基础原理的前沿探索，成长为驱动新一轮量子技术革命的核心引擎。在量子计算方面，NISQ时代的量子处理器已初步展示量子优势，容错量子计算的实验路线日益清晰；在量子通信方面，QKD已商业化部署，量子网络和量子卫星通信正在构建全球量子互联网的基础设施；在量子精密测量方面，量子传感器已突破经典极限，在时频标准、重力测量和磁场成像中展现应用潜力。

本文通过SYLVA本体论框架，揭示了量子光学与量子信息实验分支与其他SYLVA学科模块的十八项交叉联系，确立了该领域在TOE-SYLVA统一知识体系中的枢纽地位。面对可扩展性、噪声控制和工程化等挑战，该领域需要与量子材料科学、集成光子学、量子控制理论等交叉领域深度融合，方能实现从实验室到产业化的跨越。

量子光学与量子信息实验的未来充满希望，但也充满挑战。正如费曼所言："自然不是经典的，该死的，如果你想模拟自然，你最好把它变成量子力学的。"这一预言正在通过全球量子科学家的实验努力逐步成为现实。
