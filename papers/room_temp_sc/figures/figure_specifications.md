## General Figure Guidelines (Nature/Science/PRL)

### Technical Requirements
1. **Resolution:** Minimum 300 dpi for raster images
2. **Formats:** PDF (preferred), EPS, PNG, TIFF
3. **Color mode:** RGB for online, CMYK for print (if applicable)
4. **Font embedding:** All fonts must be embedded

### Style Guidelines
1. **Consistency:** All figures should use consistent colors, fonts, and styles
2. **Clarity:** Lines should be thick enough to remain visible at final size
3. **Labels:** All axes must have clear labels with units
4. **Legends:** Include legends where multiple data series are shown
5. **Scales:** Use appropriate linear/logarithmic scales

### Accessibility
1. **Color blindness:** Use colorblind-friendly palettes (avoid red-green combinations)
2. **Contrast:** Ensure sufficient contrast between elements
3. **Alternative text:** Provide descriptive captions
