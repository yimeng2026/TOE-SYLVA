## 快速开始

### 生成 PDF

```bash
# 安装依赖
pip install reportlab

# 运行生成脚本
python generate_pdf.py
```

### 查看设计文档

所有设计文档均为 Markdown 格式，可直接阅读或使用 Markdown 编辑器查看。
