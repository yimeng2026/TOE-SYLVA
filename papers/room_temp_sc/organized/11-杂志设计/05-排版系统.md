## 1.3 页边距系统（非对称设计）

### 非对称页边距哲学
外侧边距 > 内侧边距，营造翻阅时的"呼吸感"与动态视觉张力。

| 边距位置 | 尺寸 | 设计意图 |
|----------|------|----------|
| **天头 (Top)** | 15mm | 预留页眉+呼吸空间 |
| **地脚 (Bottom)** | 20mm | 预留页码+脚注区 |
| **切口 (外侧)** | 18mm | 翻阅时的手指持握区 |
| **订口 (内侧)** | 12mm | 装订损耗补偿 |

### 非对称比例
```
黄金比例变体: 外侧:内侧 = 1.5:1 (18mm:12mm)
视觉上营造"向右翻开下一页"的暗示动势
```

### 出血规范

| 项目 | 规格 | 说明 |
|------|------|------|
| **标准出血** | 3mm | 四边各延伸3mm |
| **跨页中缝** | 5mm | 避免装订后内容被遮挡 |
| **重要信息安全区** | 距成品边≥5mm | 文字、关键图像 |
| **满版图片** | 延伸至出血线外3mm | 确保裁切后无白边 |

## 2.5 图表样式（数据可视化美学）

### 图表设计原则
- **科技感：** 霓虹线条、发光效果
- **可读性：** 高对比度、清晰标注
- **网聊风：** 数据点用emoji标记

### 折线图规范

```
视觉元素:
- 网格线: 1px dotted #E8E8ED
- 数据线: 2px solid #B829DD (霓虹紫)
- 数据点: 4pt圆点，#00D9FF填充
- 峰值标记: 信号粉高亮 + ⚡emoji
- 坐标轴: 1px solid #4A4A55
- 标注字体: 8pt JetBrains Mono

背景:
- 渐变填充: #B829DD 5% → transparent
```

### 柱状图规范

```
视觉元素:
- 柱体: 3D效果，顶部#00D9FF，侧面#B829DD
- 柱间距: 1:0.6 (柱宽:间距)
- 最大值柱: 添加✨发光效果
- 数值标签: 柱顶8pt，JetBrains Mono
- 基线: 1px solid #0A0A0F
```

### 饼图/环形图规范

```
视觉元素:
- 环形宽度: 30%半径
- 区块边框: 2px #0A0A0F
- 主色: 霓虹紫 → 电光蓝 → 信号粉渐变
- 悬停效果(数字版): 扇形外扩 + 发光
- 图例: 右侧垂直排列，8pt字体
```

### 科学示意图规范

```
风格特征:
- 线条: 1.5px，端点圆角
- 粒子: 霓虹紫圆点，#00D9FF光晕
- 箭头: 电光蓝，三角形箭头
- 标注框: 仿代码注释样式
- 背景: 轻微点阵纹理 (#E8E8ED 10%)
```

# 第三部分：印刷工艺规范

## 3.1 特种纸张建议

### 封面用纸

| 类型 | 规格 | 特性 | 适用场景 |
|------|------|------|----------|
| **触感膜卡纸** | 300g | 丝绒触感，抗指纹 | 标准版封面 |
| **镭射银卡** | 250g | 彩虹折射效果 | 限量版、特别刊 |
| **黑卡纸** | 350g | 深沉黑色基底 | 典藏版 |

### 内页用纸

| 页面类型 | 纸张规格 | 特性 |
|----------|----------|------|
| **正文页** | 100g 道林纸 | 温和米色，阅读舒适 |
| **插页/专题** | 120g 铜版纸 | 高白度，图像还原佳 |
| **硫酸纸页** | 90g 硫酸纸 | 半透明，层叠效果 |
| **金句卡片** | 250g 白卡 | 可撕下，硬度适中 |

### 纸张搭配方案

```
标准版结构:
- 封面: 触感膜300g + 局部UV
- P1-8: 120g铜版纸 (开篇彩页)
- P9-88: 100g道林纸 (正文)
- P89-96: 120g铜版纸 (封底彩页)

限量版结构:
- 封面: 镭射银卡 + 全面UV
- 间隔插页: 硫酸纸 (每16页插入1张)
- 可撕页: 种子纸 (环保理念)
```

# 第五部分：文件交付规范

## 5.1 印前文件标准

### 文件格式
```
印刷文件: PDF/X-4 (支持透明度和图层)
色彩模式: CMYK + 专色通道
分辨率: 图像350dpi，矢量元素无限
出血: 包含3mm出血区域
字体: 全部嵌入或转曲

分层结构:
- Layer 1: 文字 (黑色K100)
- Layer 2: 图像 (CMYK)
- Layer 3: 专色标记 (霓虹紫)
- Layer 4: 工艺标注 (烫金/UV等)
```

### 工艺标注规范

```
烫金区域: 使用专色"Foil"标记，100%覆盖
UV区域: 使用专色"SpotUV"标记
压凹区域: 使用专色"Deboss"标记，配合虚线框
夜光区域: 使用专色"GlowInk"标记

标注图层命名:
- [Foil] 封面标题
- [SpotUV] 封面图像高光
- [Deboss] Logo压凹
```

## 5.2 打样与校色

### 打样流程
```
1. 数码打样: 确认排版和内容
2. 色彩打样: 确认专色和CMYK还原
3. 工艺打样: 确认烫金/UV等效果
4. 签字样: 最终确认，签字后上机
```

### 色彩标准
```
- 打样标准: Fogra39或GRACoL2006
- 光源: D50标准光源
- 观测角度: 45°/0°
- Delta E容忍度: ≤2.0
```
