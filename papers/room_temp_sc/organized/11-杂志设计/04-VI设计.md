## 2. Logo 系统

### 2.1 主Logo — 横版 (Horizontal)

```
╔══════════════════════════════════════════════════════════╗
║                                                          ║
║     ░▒▓ Sʏʟᴠᴀ ▓▒░          [TECH × CULTURE]              ║
║                                                          ║
║     比例: 4:1  |  最小使用尺寸: 120px宽                  ║
║                                                          ║
╚══════════════════════════════════════════════════════════╝
```

**设计说明:**
- 主文字采用特殊Unicode小字符「Sʏʟᴠᴀ」，营造隐秘/亚文化感
- 左侧故障纹理块「░▒▓ ▓▒░」为品牌符号，可独立使用
- 副标「[TECH × CULTURE]」使用方括号包裹，强调网聊/代码风格
- 主色渐变：霓虹紫(#B829DD) → 电光蓝(#00D9FF)

### 2.2 副Logo — 竖版 (Vertical)

```
╔═══════════╗
║   ░▒▓     ║
║   ▓▒░     ║
║           ║
║  Sʏʟᴠᴀ   ║
║  ─────    ║
║  MAGAZINE ║
║           ║
╚═══════════╝
     1:1.5
```

**设计说明:**
- 故障符号垂直排列，强化视觉冲击力
- 「─────」分割线呼应终端命令行美学
- 适用于社交媒体头像、杂志封面竖版、海报

### 2.4 Logo 安全空间

```
        ←—— 0.5X ——→
    ↑   ┌─────────────┐   ↑
    │   │             │   │
 0.5X   │   [LOGO]    │ 0.5X
    │   │             │   │
    ↓   └─────────────┘   ↓
        ←—— 0.5X ——→
        
X = Logo高度
```

### 2.5 Logo 禁用规范

| ❌ 错误用法 | 说明 |
|------------|------|
| 拉伸变形 | 禁止非等比缩放 |
| 颜色替换 | 禁止非品牌色使用 |
| 添加阴影 | 禁止投影/描边/渐变覆盖 |
| 背景不清 | 禁止在复杂背景上使用 |
| 旋转倾斜 | 禁止旋转超过5° |
