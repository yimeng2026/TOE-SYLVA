## 1. Koide 关系与质量谱的深层结构

### 1.1 Koide 关系的精确表述

**定义 1.1.1** (Koide 质量关系)
对三代带电轻子 $(e, \mu, \tau)$，其质量满足以下精确关系：

$$K_\ell = \frac{m_e + m_\mu + m_\tau}{(\sqrt{m_e} + \sqrt{m_\mu} + \sqrt{m_\tau})^2} = \frac{2}{3}$$

实验验证：
- $m_e = 0.510998950$ MeV
- $m_\mu = 105.6583755$ MeV
- $m_\tau = 1776.86$ MeV

$$K_\ell^{\text{exp}} = 0.666659 \approx \frac{2}{3}$$

精度达 $10^{-5}$，远超偶然巧合的可能性。

**定理 1.1.1** (Koide 关系的代数结构)
Koide 关系等价于要求质量矩阵的平方根矩阵具有特定结构：

设 $M = \text{diag}(m_e, m_\mu, m_\tau)$，则存在酉矩阵 $U$ 使得：

$$U^\dagger \sqrt{M} U = \sqrt{m_0} \cdot \mathbf{1} + \delta M$$

其中 $m_0 = (m_e + m_\mu + m_\tau)/3$，$\delta M$ 的迹为零且满足特定约束。

*证明*: 由 Koide 关系的等价形式：
$$\sqrt{m_e} + \sqrt{m_\mu} + \sqrt{m_\tau} = \sqrt{\frac{3}{2}(m_e + m_\mu + m_\tau)}$$

两边平方并整理，可得：
$$\sum_{i<j} \sqrt{m_i m_j} = \frac{1}{4}(m_e + m_\mu + m_\tau)$$

这表明 $\sqrt{m_i}$ 构成等差数列的某种变体。设 $x_i = \sqrt{m_i}$，则约束为：
$$4(x_1 x_2 + x_2 x_3 + x_3 x_1) = (x_1^2 + x_2^2 + x_3^2)$$

这是二次曲面上的约束。其解空间对应 $SO(3)$ 的特定轨道。$\square$

### 1.2 Koide 关系的几何解释

**定义 1.2.1** (质量空间)
定义三代质量空间为 $\mathbb{R}^3_+$，坐标 $(m_1, m_2, m_3)$。Koide 关系在此空间中定义一个 2 维曲面：

$$\mathcal{K} = \{(m_1, m_2, m_3) \in \mathbb{R}^3_+ \mid K(m_1, m_2, m_3) = 2/3\}$$

**定理 1.2.1** (Koide 曲面的拓扑)
Koide 曲面 $\mathcal{K}$ 是拓扑球面 $S^2$（在适当紧化下），其参数化可由球极投影给出：

$$m_1 = m_0 (1 + r \sin\theta \cos\phi)^2$$
$$m_2 = m_0 (1 + r \sin\theta \sin\phi)^2$$
$$m_3 = m_0 (1 + r \cos\theta)^2$$

其中 $r = \sqrt{3/2}$，$m_0$ 是尺度参数。

*证明*: 将 Koide 关系改写为 $\sqrt{m_i}$ 的约束。令 $x_i = \sqrt{m_i/m_0}$，则约束变为：
$$(x_1 + x_2 + x_3)^2 = \frac{3}{2}(x_1^2 + x_2^2 + x_3^2)$$

展开并整理：
$$x_1^2 + x_2^2 + x_3^2 + 2(x_1 x_2 + x_2 x_3 + x_3 x_1) = \frac{3}{2}(x_1^2 + x_2^2 + x_3^2)$$

$$2(x_1 x_2 + x_2 x_3 + x_3 x_1) = \frac{1}{2}(x_1^2 + x_2^2 + x_3^2)$$

这定义了 $\mathbb{R}^3$ 中的一个锥面。通过尺度变换 $x_i \to x_i/(x_1 + x_2 + x_3)$，可将其投影到球面上。$\square$

### 1.3 夸克质量的 Koide 推广

**定义 1.3.1** (夸克扇区的 Koide 参数)
对夸克质量，定义 Koide-like 参数：

$$K_q^{(u)} = \frac{m_u + m_c + m_t}{(\sqrt{m_u} + \sqrt{m_c} + \sqrt{m_t})^2}$$
$$K_q^{(d)} = \frac{m_d + m_s + m_b}{(\sqrt{m_d} + \sqrt{m_s} + \sqrt{m_b})^2}$$

实验值：
- $K_q^{(u)} \approx 0.62$（接近但不等于 2/3）
- $K_q^{(d)} \approx 0.73$（偏离较大）

**定理 1.3.1** (Koide 参数的层级修正)

$$K_q^{\text{eff}} = K_q^{(0)} \left(1 - \frac{\alpha_s(\mu)}{\pi} \cdot C_F \cdot f(\mu/m_t)\right)$$

*证明*: 夸克质量在 QCD 中通过重整化群跑动。在 $	ext{MS}$ 方案下：
$$\frac{d \ln m_q}{d \ln \mu} = -2\gamma_m(\alpha_s) = -2\frac{\alpha_s}{\pi} C_F + O(\alpha_s^2)$$

质量的跑动改变了 Koide 参数的有效值。将跑动质量代入 Koide 公式，得到能标依赖的修正。在 GUT 能标处，跑动效应减弱，$K_q$ 可能趋近于 2/3。$\square$

$$g_{GUT}^2 = g_3^2(M_{GUT}) = g_2^2(M_{GUT}) = \frac{5}{3}g_1^2(M_{GUT})$$

## 5. 无量纲常数的 Galois 理论

### 5.1 常数域的 Galois 群

**定义 5.1.1** (常数域)
定义无量纲常数域 $\mathbb{F}_C$ 为包含所有标准模型无量纲常数的最小数域：

$$\mathbb{F}_C = \mathbb{Q}(\alpha, \alpha_w, \alpha_s, \theta_{QCD}, y_e, y_\mu, y_\tau, y_u, y_d, y_c, y_s, y_t, y_b, \theta_{ij}^{CKM}, \theta_{ij}^{PMNS}, \delta_{CKM}, \delta_{CP})$$

**定理 5.1.1** (常数域的超越次数)
常数域 $\mathbb{F}_C$ 的超越次数至少为 15（对应 15 个基本常数）：

$$\text{trdeg}(\mathbb{F}_C/\mathbb{Q}) \geq 15$$

若所有常数可由单一超越数生成（如弦理论期望），则：

$$\text{trdeg}(\mathbb{F}_C/\mathbb{Q}) = 1$$

**证明*: 15 个基本常数在实验上相互独立（无已知代数关系）。因此它们代数独立，超越次数至少为 15。若存在统一理论（如弦理论），所有常数由模空间的真空期望值决定，这些期望值由模场代数函数的零点给出。若模空间连通，所有常数可能由单一参数（如 dilaton）生成。$\square$

### 5.2 Galois 群与常数变换

**定义 5.2.1** (常数域的 Galois 群)
$\text{Gal}(\mathbb{F}_C/\mathbb{Q})$ 是常数域的自同构群，保持 $\mathbb{Q}$ 不变。

**定理 5.2.1** (Galois 群与物理对称性)
若常数域的 Galois 群非平凡，则存在常数置换对应物理等价理论：

$$\sigma \in \text{Gal}(\mathbb{F}_C/\mathbb{Q}): \quad \alpha \mapsto \sigma(\alpha), \quad \alpha_w \mapsto \sigma(\alpha_w)$$

这些置换对应于宇宙的不同"分支"或"景观"中的等价类。

*注*: 在弦景观中，不同真空对应不同的常数集合。Galois 群可能作用于景观的离散参数空间上，将不同真空关联起来。

### 8.2 常数预测的数学框架

**定义 8.2.1** (常数预测映射)
定义预测映射 $P: \mathcal{T} \to \mathbb{R}^{15}$，从理论空间 $\mathcal{T}$ 到常数空间：

$$P(\mathcal{T}_i) = (C_1(\mathcal{T}_i), C_2(\mathcal{T}_i), \ldots, C_{15}(\mathcal{T}_i))$$

**定理 8.2.1** (预测映射的纤维结构)
预测映射的纤维 $P^{-1}(C)$ 对应于产生相同常数集的所有理论：

$$P^{-1}(C) = \{\mathcal{T}_i \mid P(\mathcal{T}_i) = C\}$$

若纤维是离散的（有限个理论），则常数可预测；若纤维是连续的，则常数依赖于连续参数（模），需额外原理选择。

*注*: 在弦理论中，纤维通常是离散的（通量量子化），但数量巨大（$\sim 10^{500}$）。在圈量子引力中，纤维可能是连续的（由于 Barbero-Immirzi 参数的连续选择）。
