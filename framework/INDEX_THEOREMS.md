## 🧮 一、数学定理

### 1.1 范畴论语义 (Category Theory)

#### 定理 1.1.1: 因果范畴的完备性
- **文档**: [[doc:05]]
- **陈述**: 任何有限因果网络 G = (V, E) 可唯一地嵌入一个因果范畴 𝒞(G)，其中对象是 V 的幂集 2^V，态射对应因果依赖关系，该嵌入保持所有路径信息。
- **证明概要**: 构造从 G 到 𝒞(G) 的自由函子，验证泛性质。
- **相关定理**: Yoneda引理，下降定理
- **交叉引用**: [[theorem:Yoneda引理在因果网络中的应用]]

#### 定理 1.1.2: 层到整体 (Sheaf to Global)
- **文档**: [[doc:05]]
- **陈述**: 设 ℱ 是因果范畴 𝒞 上的层，则全局截面函子 Γ(𝒞, ℱ) = lim←_{e∈𝒞} ℱ(e) 存在当且仅当局部相容性条件满足。
- **物理解释**: 这是因果网络中"从局部物理到全局涌现"的数学表述。
- **相关定理**: 谱序列收敛定理

#### 定理 1.1.3: Yoneda引理在因果网络中的应用
- **文档**: [[doc:05]]
- **陈述**: 对于因果范畴 𝒞，Yoneda嵌入 y: 𝒞 → [𝒞^op, Set] 将每个事件 e 映射为其可观测性质的总和：y(e) = Hom(-, e)。
- **物理意义**: 一个事件的"本质"由其与其他所有事件的关系网络完全确定。
- **相关概念**: 可表函子，万有性质

### 1.2 同调代数 (Homological Algebra)

#### 定理 1.2.1: 电荷量子化的同调解释
- **文档**: [[doc:05]]
- **陈述**: 在因果网络中，电荷必须量子化：Q = n·e₀，其中 n ∈ ℤ。这是因为 Q ∈ H²(𝒞, ℤ) ≅ Hom(H₂(𝒞), ℤ)。
- **证明概要**: 考虑U(1)规范场在因果网络中的联络1-形式 A，其场强 F = dA 满足 ∫_{Σ₂} F = 2πn ∈ 2πℤ。由de Rham定理，积分周期条件强制其来自整系数上同调。
- **相关定理**: de Rham定理，Gauss-Bonnet定理
- **交叉引用**: [[theorem:谱序列与涌现]]

#### 定理 1.2.2: 谱序列与涌现
- **文档**: [[doc:05]]
- **陈述**: 因果范畴 𝒞 上的层 ℱ 有谱序列 E₂^{p,q} = H^p(𝒞, ℋ^q(ℱ)) ⇒ H^{p+q}(Tot(ℱ))，其中微分 d_r: E_r^{p,q} → E_r^{p+r,q-r+1} 编码了不同尺度间的耦合。
- **物理解释**: E₂页对应局域物理，E_∞页对应全局涌现，微分对应从局域到全局的"障碍"。
- **应用领域**: 重整化群，有效场论

#### 定理 1.2.3: 障碍类的计算
- **文档**: [[doc:05]]
- **陈述**: 给定扩张问题 0 → A → B → C → 0，其障碍类位于 Ob(ℱ) ∈ Ext²_𝒞(C, A)，扩张存在当且仅当 Ob(ℱ) = 0。
- **物理应用**: 规范对称性的异常对应 Ext 群的非零元素。
- **相关定理**: 长正合序列，连接同态

### 1.3 非交换几何 (Noncommutative Geometry)

#### 定理 1.3.1: 因果网络代数的K-理论
- **文档**: [[doc:27]], [[doc:05]]
- **陈述**: 因果网络代数 𝒜(G) 的K-群分类投影类：K₀(𝒜(G)) ≅ ℤ^{|V|} ⊕ Torsion，其中自由部分对应顶点，扭部分对应网络拓扑。
- **物理意义**: 粒子种类对应K-群的生成元。
- **相关定理**: Bott周期性，指标定理

#### 定理 1.3.2: 谱作用原理 (Spectral Action Principle)
- **文档**: [[doc:27]], [[doc:05]]
- **陈述**: 作用量由谱三元组的谱迹给出：S_Λ[𝒟] = Tr(f(𝒟²/Λ²))，展开后得到 Einstein-Hilbert作用 + 宇宙学常数 + 标准模型拉氏量。
- **结果**: 引力与标准模型自然涌现。
- **公式**: [[eq:谱作用展开]]

#### 定理 1.3.3: 从非交换几何导出标准模型
- **文档**: [[doc:27]], [[doc:05]]
- **陈述**: 设有限代数 𝒜_F = ℍ ⊕ ℂ（四元数+复数），则：
  1. 自同构群给出规范对称性：SU(3) × SU(2) × U(1)
  2. Hilbert空间 ℋ_F 编码三代费米子
  3. Dirac算子 𝒟_F 产生Yukawa耦合
- **证明概要**: 验证Boeijink-van Suijlekom分类定理的条件。
- **相关定理**: Connes重建定理

#### 定理 1.3.4: Connes重建定理
- **文档**: [[doc:27]]
- **陈述**: 给定谱三元组 (𝒜, ℋ, 𝒟)，若满足特定公理（定向、维数、一阶、庞加莱对偶、实结构），则可重建紧自旋流形及其黎曼度规。
- **意义**: 从纯代数数据恢复几何。
- **交叉引用**: [[theorem:因果性在非交换几何中的表述]]

#### 定理 1.3.5: 因果性在非交换几何中的表述
- **文档**: [[doc:27]], [[doc:05]]
- **陈述**: 在谱三元组 (𝒜, ℋ, 𝒟) 中，因果结构由序关系编码：a ≼ b ⟺ ⟨aξ, ξ⟩ ≤ ⟨bξ, ξ⟩, ∀ξ ∈ ℋ⁺，其中 ℋ⁺ 是 𝒟 正频部分定义的圆锥。
- **应用**: 非交换时空的因果结构定义。

### 1.4 拓扑量子场论 (TQFT)

#### 定理 1.4.1: 陈-西蒙斯配分函数
- **文档**: [[doc:31]], [[doc:05]]
- **陈述**: Z_k(M) = ∫ 𝒟A e^{i S_{CS}[A]} = ∑_ρ |Hom(π₁(M), G)_ρ| · e^{2πi·CS(ρ)}，其中求和遍历平坦联络的同痕类。
- **物理解释**: 拓扑不变量仅依赖于流形的基本群。
- **相关定理**: WRT不变量

#### 定理 1.4.2: Witten-Reshetikhin-Turaev不变量
- **文档**: [[doc:31]], [[doc:05]]
- **陈述**: WRT_k(M) = η^{-b₁(M)} ∑_ρ (∏_v Tet(v))/(∏_e dim_q(j_e))，其中 η 是归一化因子，Tet(v) 是量子6j符号，dim_q 是量子维数。
- **结果**: 量子纽结不变量（Jones多项式的推广）自然涌现。
- **相关定理**: Jones多项式，HOMFLY多项式

#### 定理 1.4.3: TQFT与量子引力的联系
- **文档**: [[doc:31]], [[doc:11]]
- **陈述**: 圈量子引力（LQG）的希尔伯特空间可构造为TQFT的态空间：ℋ_{LQG} = ⊕_Γ ⊗_e V_{j_e} ⊗_v Inv(i_v)。
- **对应关系**: 
  - 自旋网络 ↔ TQFT基态
  - 面积算子 Â(S) = 8πγℓ_P² ∑_{p∈S∩Γ} √(j_p(j_p+1))
  - 体积算子 V̂(R) = ℓ_P³ ∑_{v∈R} V̂_v
- **交叉引用**: [[doc:11]] 量子引力

#### 定理 1.4.4: 全息原理的TQFT表述
- **文档**: [[doc:31]], [[doc:14]]
- **陈述**: 对于带边界的(d+1)维区域 M，边界上的TQFT编码内部信息：Z(M) = ⟨ψ_{∂M}| 𝒪_{bulk} |ψ_{∂M}⟩，其中 |ψ_{∂M}⟩ ∈ Z(∂M) 是边界态。
- **应用**: AdS/CFT对应的形式化。

### 1.5 同伦类型论 (HoTT)

#### 定理 1.5.1: 单值公理 (Univalence Axiom)
- **文档**: [[doc:05]]
- **陈述**: (A =_𝒰 B) ≃ (A ≃ B)，类型的相等等价于等价（同伦等价）。
- **物理意义**: 物理上等价的理论（对偶性）在类型论中是同一理论的不同的证明。
- **应用**: 弦理论对偶性的形式化。

#### 定理 1.5.2: 路径积分作为HIT
- **文档**: [[doc:05]]
- **陈述**: 对于场构型空间 ℱ，路径积分测度可由高阶归纳类型构造，积分值为 Z = ∑_{[ϕ]∈π₀(ℱ)} e^{iS[ϕ]}/|Aut(ϕ)|。
- **相关概念**: 高阶归纳类型(HIT)，商类型

### 1.6 可积系统 (Integrable Systems)

#### 定理 1.6.1: KdV单孤子解
- **文档**: [[doc:32]]
- **陈述**: KdV方程具有单孤子解 u(x,t) = 2κ² sech²[κ(x - 4κ²t - x₀)]，其中 κ > 0 为谱参数，x₀ 为初始位置。
- **证明**: 行波假设 u(x,t) = f(ξ)，ξ = x - ct，代入KdV方程求解ODE。
- **公式**: [[eq:KdV单孤子解]]

#### 定理 1.6.2: Miura变换
- **文档**: [[doc:32]]
- **陈述**: 若 v(x,t) 满足修正KdV方程 ∂_t v - 6v²∂_x v + ∂_x³ v = 0，则通过 Miura变换 u = v² + ∂_x v 得到的 u 满足KdV方程。
- **意义**: 将KdV与mKdV联系，揭示与薛定谔方程的深刻联系。
- **相关定理**: GGKM逆散射方法

#### 定理 1.6.3: GGKM逆散射方法
- **文档**: [[doc:32]]
- **陈述**: KdV方程可通过逆散射变换求解：给定初值 u(x,0)，先求薛定谔方程 -ψ_xx + uψ = λψ 的散射数据，再演化散射数据，最后逆变换重构 u(x,t)。
- **关键步骤**: 直接散射 → 时间演化 → 逆散射
- **相关定理**: Lax对表示

#### 定理 1.6.4: Lax对表示
- **文档**: [[doc:32]]
- **陈述**: KdV方程可表示为Lax方程 ∂_t L = [L, A]，其中 L = -∂_x² + u 为薛定谔算子，A = 4∂_x³ - 3(u∂_x + ∂_x u) 为辅助算子。
- **意义**: 等谱演化，无穷多守恒量。
- **相关定理**: 零曲率方程

#### 定理 1.6.5: Sato理论
- **文档**: [[doc:32]]
- **陈述**: KP/KdV方程的解可通过Sato构造从无限Grassmannian得到，τ函数对应Plücker坐标，解空间具有代数几何结构。
- **应用**: 精确解的分类，代数曲线方法。

### 1.7 随机矩阵 (Random Matrix Theory)

#### 定理 1.7.1: Wigner半圆律
- **文档**: [[doc:29]]
- **陈述**: 对于N×N厄米随机矩阵，当N → ∞时，本征值密度收敛于半圆分布 ρ(λ) = (2/πR²)√(R² - λ²)。
- **适用条件**: 独立同分布矩阵元，有限方差。
- **普适性**: 与具体分布无关（普适性类）。

#### 定理 1.7.2: BGS猜想
- **文档**: [[doc:29]]
- **陈述**: 量子混沌系统的能级统计与随机矩阵理论的预测一致（GOE/GUE/GSE对应不同对称性类）。
- **物理解释**: 混沌系统的谱关联由普适性决定。

#### 定理 1.7.3: Montgomery-Odlyzko定律
- **文档**: [[doc:29]]
- **陈述**: Riemann ζ函数零点的间距分布与GUE随机矩阵的本征值间距一致。
- **意义**: 数论与随机矩阵的深刻联系。
- **交叉引用**: [[doc:15]] 可计算宇宙

### 1.8 信息几何

#### 定理 1.8.1: Cramér-Rao下界
- **文档**: [[doc:30]]
- **陈述**: 对于参数估计，任何无偏估计量的协方差矩阵满足 Cov(θ̂) ≥ I(θ)^{-1}，其中 I(θ) 是Fisher信息矩阵。
- **几何解释**: 统计流形上的曲率决定了估计精度极限。

#### 定理 1.8.2: Amari对偶平坦性
- **文档**: [[doc:30]]
- **陈述**: 指数族分布具有对偶平坦结构，存在两组对偶坐标系 {θ^i} 和 {η_i} 通过Legendre变换联系，且都存在仿射坐标。
- **应用**: 自然梯度下降，统计推断优化。

## ⚛️ 二、物理定理

### 2.1 量子场论 (Quantum Field Theory)

#### 定理 2.1.1: 渐近自由
- **文档**: [[doc:03]]
- **陈述**: 非阿贝尔规范理论（如QCD）在高能（短距离）下耦合常数趋于零，夸克和胶子表现为自由粒子。
- **数学表达**: β(g) = -g³/(16π²)(11 - 2n_f/3) + O(g⁵) < 0
- **发现者**: Gross, Wilczek, Politzer (2004诺贝尔奖)
- **交叉引用**: [[theorem:重整化群方程]]

#### 定理 2.1.2: 电弱统一 (Glashow-Weinberg-Salam模型)
- **文档**: [[doc:08]]
- **陈述**: 电磁力与弱核力在高能下统一为电弱力，由SU(2)_L × U(1)_Y规范群描述，通过希格斯机制自发对称性破缺到低能的U(1)_em。
- **预言**: W±, Z⁰玻色子质量，中性流过程。
- **实验验证**: CERN UA1/UA2实验(1983)。

#### 定理 2.1.3: 颜色禁闭
- **文档**: [[doc:03]]
- **陈述**: 夸克和胶子被禁闭在色单态强子内，无法作为自由粒子观测。
- **证据**: 线性禁闭势，喷注现象，格点QCD计算。
- **未解决问题**: 严格数学证明。

### 2.2 统一场论 (Unified Field Theory)

#### 定理 2.2.1: GUT耦合统一
- **文档**: [[doc:10]]
- **陈述**: 在SU(5)或SO(10)大统一理论中，三个规范耦合在高能标(~10¹⁵ GeV)处汇聚为单一耦合。
- **公式**: sin²θ_W(M_Z) = 3/8(1 - (55α/36π)ln(M_GUT/M_Z) + ...)
- **预言**: 质子衰变，磁单极。

#### 定理 2.2.2: 跷跷板机制 (See-Saw Mechanism)
- **文档**: [[doc:09]], [[doc:10]], [[doc:16]]
- **陈述**: 通过引入重Majorana中微子，轻中微子质量被压低为 m_ν ≈ m_D²/M_R，其中m_D是Dirac质量，M_R是重中微子质量。
- **物理解释**: 轻子数破坏，轻子生成机制。
- **交叉引用**: [[doc:09]] 中微子暴胀

#### 定理 2.2.3: SUSY代数
- **文档**: [[doc:12]]
- **陈述**: 超Poincaré代数包含反对易关系 {Q_α, Q̄_{β̇}} = 2σ^μ_{αβ̇} P_μ，其中Q_α是超荷，P_μ是四动量。
- **推论**: 超对称伙伴质量相等（未破缺时），玻色子与费米子配对。

### 2.3 量子引力 (Quantum Gravity)

#### 定理 2.3.1: Bekenstein-Hawking熵
- **文档**: [[doc:14]], [[doc:20]]
- **陈述**: 黑洞熵与其视界面积成正比：S_BH = k_B c³ A / (4Gℏ) = A / (4Gℏ) (自然单位制)。
- **意义**: 黑洞热力学，全息原理的启发。
- **微观解释**: 弦理论(D-brane)，圈量子引力(自旋网络)。
- **公式**: [[eq:Bekenstein-Hawking熵]]

#### 定理 2.3.2: Hawking辐射
- **文档**: [[doc:20]], [[doc:14]]
- **陈述**: 黑洞发出热辐射，温度 T_H = ℏ c³ / (8πGMk_B) = 1/(8πM) (自然单位制)。
- **推导**: 弯曲时空量子场论，视界处的Bogoliubov变换。
- **悖论**: 信息悖论，黑洞蒸发导致纯态到混合态的跃迁。

#### 定理 2.3.3: Page曲线
- **文档**: [[doc:14]], [[doc:20]]
- **陈述**: 对于辐射子系统，纠缠熵先增长至Page时间(~M³)，然后开始下降，最终归零（恢复幺正性）。
- **意义**: 信息悖论的解决方案。
- **新进展**: 岛屿公式(2020)。

#### 定理 2.3.4: 岛屿公式
- **文档**: [[doc:14]], [[doc:20]]
- **陈述**: S(ρ_R) = min{ext ∂I}[Area(∂I)/(4G_N) + S_bulk(ρ_{R∪I})]，其中I是岛屿区域，∂I是其边界。
- **物理解释**: 后期辐射与早期辐射之间的量子纠缠通过虫洞连接。
- **交叉引用**: [[theorem:ER=EPR猜想]]

#### 定理 2.3.5: AdS/CFT对应 (Maldacena对偶)
- **文档**: [[doc:25]], [[doc:14]]
- **陈述**: IIB型弦理论在AdS₅ × S⁵背景下等价于边界上的N=4超杨-米尔斯理论。
- **字典**: 
  - 引力物理 ↔ 边界场论
  - 弦张力 ↔ 't Hooft耦合
  - 弱耦合 ↔ 强耦合对偶
- **推广**: 全息原理的精确实现。
- **交叉引用**: [[theorem:全息原理的TQFT表述]]

#### 定理 2.3.6: ER=EPR猜想
- **文档**: [[doc:22]], [[doc:14]]
- **陈述**: 爱因斯坦-罗森桥(虫洞)与量子纠缠(EPR对)是同一现象的不同描述。
- **意义**: 几何与量子信息的深层联系。
- **支持证据**: SYK模型，简单虫洞模型。

### 2.4 弦理论 (String Theory)

#### 定理 2.4.1: T-对偶
- **文档**: [[doc:25]], [[doc:13]]
- **陈述**: 紧化在半径为R的圆上的弦等价于紧化在半径为α'/R的圆上的弦，动量模与缠绕模互换。
- **公式**: R ↔ α'/R，p ↔ w
- **物理**: 几何概念在弦理论中是模糊的（最小距离~√α'）。

#### 定理 2.4.2: S-对偶
- **文档**: [[doc:25]], [[doc:12]]
- **陈述**: IIB型弦理论在强弱耦合之间自对偶：g_s ↔ 1/g_s。
- **其他对偶**: IIA ↔ M理论(紧致化)，SO(32)杂化 ↔ I型。
- **意义**: 五种超弦理论是单一M理论的不同极限。

#### 定理 2.4.3: M理论统一
- **文档**: [[doc:25]], [[doc:11]]
- **陈述**: 五种超弦理论和11维超引力都是11维M理论在不同极限下的表现。
- **M理论**: 非微扰框架，包含M2膜和M5膜。
- **未解决问题**: M理论的完整定义（可能是矩阵理论）。

### 2.5 黑洞物理 (Black Hole Physics)

#### 定理 2.5.1: 黑洞热力学四定律
- **文档**: [[doc:20]], [[doc:14]]
- **陈述**: 
  1. 第零定律：表面引力κ在稳态视界上是常数（类比温度）
  2. 第一定律：δM = (κ/8πG)δA + ΩδJ + ΦδQ（类比能量守恒）
  3. 第二定律：δA ≥ 0（霍金面积定理，类比熵增）
  4. 第三定律：不可能通过有限过程达到κ = 0
- **意义**: 经典广义相对论与热力学的深刻类比。

#### 定理 2.5.2: 黑洞无毛定理
- **文档**: [[doc:20]]
- **陈述**: 稳态渐近平直黑洞由质量M、角动量J、电荷Q三个参数完全刻画（Kerr-Newman解）。
- **推论**: 黑洞不保留落入物质的细节信息（导致信息悖论）。
- **限制**: 假设渐近平直，可能有例外（如Q-clouds）。

### 2.6 宇宙学 (Cosmology)

#### 定理 2.6.1: 慢滚暴胀
- **文档**: [[doc:09]], [[doc:19]]
- **陈述**: 当标量场满足慢滚条件 ε = -Ḣ/H² ≪ 1，η = M_Pl² V''/V ≪ 1 时，宇宙经历指数膨胀，a(t) ∝ e^{Ht}。
- **预言**: 近似尺度不变的原初功率谱，P_ζ(k) ≈ (H/2π)²/(2εM_Pl²)。
- **交叉引用**: [[doc:09]] 中微子暴胀

#### 定理 2.6.2: 宇宙学扰动理论
- **文档**: [[doc:09]], [[doc:19]]
- **陈述**: 原初扰动可分为标量（密度）、矢量（涡旋）、张量（引力波）模式，在暴胀期间量子涨落被拉伸到经典尺度。
- **观测**: CMB各向异性，大尺度结构。

#### 定理 2.6.3: 重子生成萨哈罗夫条件
- **文档**: [[doc:19]], [[doc:10]]
- **陈述**: 产生宇宙重子不对称需要三个条件：
  1. 重子数不守恒
  2. C和CP破坏
  3. 偏离热平衡
- **机制**: 电弱重子生成，轻子生成，GUT重子生成。

### 2.7 量子信息 (Quantum Information)

#### 定理 2.7.1: 不可克隆定理
- **文档**: [[doc:17]], [[doc:22]]
- **陈述**: 不存在一个物理过程能够完美复制任意未知量子态。
- **证明**: 线性量子力学的直接结果。
- **应用**: 量子密码的安全性基础。

#### 定理 2.7.2: Bell不等式
- **文档**: [[doc:22]], [[doc:17]]
- **陈述**: 对于局域实在论，S = |E(a,b) - E(a,b') + E(a',b) + E(a',b')| ≤ 2；量子力学预言 S ≤ 2√2。
- **实验**: Aspect等(1982)， loophole-free实验(2015)。
- **意义**: 量子非定域性的定量检验。
- **公式**: [[eq:CHSH不等式]]

#### 定理 2.7.3: 量子纠错阈值定理
- **文档**: [[doc:17]], [[doc:14]]
- **陈述**: 如果物理错误率低于某个阈值(~10^{-4} - 10^{-2})，则可以通过级联纠错码实现任意可靠的量子计算。
- **意义**: 容错量子计算的理论基础。

### 2.8 暗物质与暗能量

#### 定理 2.8.1: 弗里德曼方程
- **文档**: [[doc:04]], [[doc:23]], [[doc:19]]
- **陈述**: H² = (ȧ/a)² = (8πG/3)ρ - k/a² + Λ/3，描述均匀各向同性宇宙的膨胀动力学。
- **组成**: ρ = ρ_m + ρ_r + ρ_Λ（物质+辐射+暗能量）。
- **公式**: [[eq:Friedmann方程]]

#### 定理 2.8.2: 暗物质晕密度轮廓
- **文档**: [[doc:18]], [[doc:04]]
- **陈述**: NFW轮廓 ρ(r) = ρ_s / [(r/r_s)(1 + r/r_s)²] 描述宇宙学模拟中暗物质晕的形成。
- **替代**: Einasto轮廓，Moore轮廓。
- **相关**: 自相似解，球对称坍缩。
