## 问题诊断

### 根本原因分析

当前公式:
```
e_eff = √(mean_degree/N) × (1/√dim) × √κ
α = e_eff² / (4π)
```

实际输出 α ≈ 0.0026，目标 α ≈ 0.0073，**差距约 2.8 倍**。

### 可能的解决方案

1. **全局缩放因子**
   ```
   α_corrected = C × α_current
   C ≈ 2.8 (由实验数据拟合)
   ```

2. **修正理论公式**
   ```
   α = dim × e_eff² / (4π)  # 乘以维度因子
   ```
   这将提供约 4x 的增益，使 α ≈ 0.0105 (偏差 44%)。

3. **调整连通性度量**
   ```
   e_eff ~ mean_degree^(2/3) / N^(1/3)  # 改变幂律
   ```

4. **引入网络拓扑修正**
   - 考虑聚类系数
   - 引入度分布的高阶矩

## 外推到 N→∞

基于有限尺寸标度分析，当 N → ∞ 时:

```
α(N) = α_∞ + a/N + b/N²
```

当前数据 (N=1000) 显示 α 随 N 增加而缓慢增长，预计 N→∞ 时 α_∞ ≈ 0.0030，**仍低于目标 0.0073**。
