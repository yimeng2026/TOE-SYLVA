## 第一节：范畴论基础回顾——函子、自然变换与伴随

### 1.1 范畴的公理化定义

**定义 1.1.1（范畴）**。一个**范畴** $\mathcal{C}$ 由以下数据构成：

1. **对象类** $\text{Ob}(\mathcal{C})$，其元素称为范畴的对象；
2. **态射类**，对任意 $X, Y \in \text{Ob}(\mathcal{C})$，存在集合（或类）$\text{Hom}_{\mathcal{C}}(X, Y)$，其元素称为从 $X$ 到 $Y$ 的**态射**（或箭头）；
3. **复合运算**，对任意三元组 $X, Y, Z \in \text{Ob}(\mathcal{C})$，存在映射：
   $$\circ: \text{Hom}_{\mathcal{C}}(Y, Z) \times \text{Hom}_{\mathcal{C}}(X, Y) \longrightarrow \text{Hom}_{\mathcal{C}}(X, Z)$$
   记作 $(g, f) \mapsto g \circ f$；
4. **单位态射**，对每个对象 $X \in \text{Ob}(\mathcal{C})$，存在**恒等态射** $\text{id}_X \in \text{Hom}_{\mathcal{C}}(X, X)$。

这些数据需满足以下公理：

- **结合律**：对任意 $f: W \to X$, $g: X \to Y$, $h: Y \to Z$，有 $(h \circ g) \circ f = h \circ (g \circ f)$；
- **单位律**：对任意 $f: X \to Y$，有 $f \circ \text{id}_X = f = \text{id}_Y \circ f$。

```
┌─────────────────────────────────────────────────────────────────┐
│                    范畴的基本结构                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    对象: X, Y, Z, ...                                          │
│         ●          ●          ●                                  │
│         │          │          │                                 │
│         │ f        │ g        │ h                               │
│         ↓          ↓          ↓                                 │
│         ●          ●          ●                                  │
│         Y          Z          W                                  │
│                                                                 │
│    态射复合: X ──f──→ Y ──g──→ Z  等价于  X ──g∘f──→ Z          │
│                                                                 │
│    恒等态射: id_X: X → X,  满足 f ∘ id_X = f = id_Y ∘ f         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 函子：范畴间的结构保持映射

**定义 1.2.1（函子）**。设 $\mathcal{C}$ 和 $\mathcal{D}$ 为范畴。一个**函子** $F: \mathcal{C} \to \mathcal{D}$ 由以下数据构成：

1. **对象映射**：对每个 $X \in \text{Ob}(\mathcal{C})$，指定对象 $F(X) \in \text{Ob}(\mathcal{D})$；
2. **态射映射**：对每个 $f \in \text{Hom}_{\mathcal{C}}(X, Y)$，指定态射 $F(f) \in \text{Hom}_{\mathcal{D}}(F(X), F(Y))$。

满足以下条件：
- **保持恒等**：$F(\text{id}_X) = \text{id}_{F(X)}$；
- **保持复合**：$F(g \circ f) = F(g) \circ F(f)$。

函子可分为**协变函子**（covariant）与**反变函子**（contravariant）。反变函子反转箭头方向，即 $F(f: X \to Y): F(Y) \to F(X)$。

**例 1.2.2（遗忘函子）**。设 $\text{Grp}$ 为群范畴，$\text{Set}$ 为集合范畴。遗忘函子 $U: \text{Grp} \to \text{Set}$ 将群 $(G, \cdot)$ 映射到底层集合 $G$，将群同态映射为底层集合映射。

**例 1.2.3（同伦函子）**。基本群函子 $\pi_1: \text{Top}_* \to \text{Grp}$ 从带基点的拓扑空间范畴映射到群范畴，将空间 $(X, x_0)$ 映射到其基本群 $\pi_1(X, x_0)$。

```
┌─────────────────────────────────────────────────────────────────┐
│                       函子的结构                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    协变函子 F: C → D                                           │
│                                                                 │
│    C              D                                              │
│    ●───f───→●    ⇒    ●───F(f)───→●                           │
│    X        Y         F(X)      F(Y)                           │
│    │        │         │          │                             │
│    │  g∘f   │         │  F(g∘f)  │  = F(g)∘F(f)                │
│    ↓        ↓         ↓          ↓                             │
│    ●        ●         ●          ●                             │
│    Z        W         F(Z)      F(W)                           │
│                                                                 │
│    反变函子 F: C^op → D   (箭头方向反转)                        │
│                                                                 │
│    C              D                                              │
│    ●───f───→●    ⇒    ●←──F(f)───●                            │

```

**定理 1.2.4（Yoneda 引理）**。设 $\mathcal{C}$ 为局部小范畴，$F: \mathcal{C}^{\text{op}} \to \text{Set}$ 为反变函子。则对任意 $X \in \text{Ob}(\mathcal{C})$，存在自然同构：
$$\text{Nat}(y_X, F) \cong F(X)$$
其中 $y_X = \text{Hom}_{\mathcal{C}}(-, X)$ 为可表函子（representable functor），$\text{Nat}$ 表示自然变换的集合。

Yoneda 引理的深刻意义在于：对象 $X$ 完全由其与其他所有对象的关系（即 $y_X$）所决定。这为"结构决定本质"的数学哲学提供了严格表述。

### 1.3 自然变换：函子间的映射

**定义 1.3.1（自然变换）**。设 $F, G: \mathcal{C} \to \mathcal{D}$ 为两个函子。一个**自然变换** $\alpha: F \Rightarrow G$ 是一族态射 $(\alpha_X: F(X) \to G(X))_{X \in \text{Ob}(\mathcal{C})}$，使得对任意 $f: X \to Y$ 在 $\mathcal{C}$ 中，下图交换：

$$
\begin{array}{ccc}
F(X) & \xrightarrow{\alpha_X} & G(X) \\
F(f) \downarrow & & \downarrow G(f) \\
F(Y) & \xrightarrow{\alpha_Y} & G(Y)
\end{array}
$$

即 $G(f) \circ \alpha_X = \alpha_Y \circ F(f)$。

自然变换的交换性条件捕捉了"自然性"的本质：无论选择何种路径（先变换后映射，还是先映射后变换），结果一致。

```
┌─────────────────────────────────────────────────────────────────┐
│                    自然变换的交换图                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                    自然性条件                                    │
│                                                                 │
│           F(X) ───────α_X───────→ G(X)                        │
│             │                       │                           │
│             │ F(f)                  │ G(f)                        │
│             ↓                       ↓                           │
│           F(Y) ───────α_Y───────→ G(Y)                        │
│                                                                 │
│           G(f) ∘ α_X = α_Y ∘ F(f)   （交换性）                  │
│                                                                 │
│    直观理解: 无论走哪条路径，结果相同                            │
│              ↓                                                   │
│            F(X) ──F(f)──→ F(Y) ──α_Y──→ G(Y)                   │
│              │                                        ↑        │
│              │α_X                                      │        │
│              ↓            =                            │        │
│            G(X) ─────────G(f)───────────→ G(Y)                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**定义 1.3.2（函子范畴）**。设 $\mathcal{C}, \mathcal{D}$ 为范畴。从 $\mathcal{C}$ 到 $\mathcal{D}$ 的所有函子构成**函子范畴** $[\mathcal{C}, \mathcal{D}]$（或 $\mathcal{D}^{\mathcal{C}}$），其对象为函子，态射为自然变换。

### 1.4 伴随函子：普遍构造的对偶性

**定义 1.4.1（伴随）**。设 $F: \mathcal{C} \to \mathcal{D}$ 和 $G: \mathcal{D} \to \mathcal{C}$ 为两个函子。称 $(F, G)$ 构成**伴随对**（adjunction），记作 $F \dashv G$，如果存在自然的双射：
$$\Phi_{X,Y}: \text{Hom}_{\mathcal{D}}(F(X), Y) \cong \text{Hom}_{\mathcal{C}}(X, G(Y))$$
对任意 $X \in \text{Ob}(\mathcal{C})$, $Y \in \text{Ob}(\mathcal{D})$ 成立。

在此情形下，$F$ 称为 $G$ 的**左伴随**（left adjoint），$G$ 称为 $F$ 的**右伴随**（right adjoint）。

伴随关系可以等价地由**单位**（unit）和**余单位**（counit）刻画：
- **单位**：自然变换 $\eta: \text{id}_{\mathcal{C}} \Rightarrow G \circ F$
- **余单位**：自然变换 $\varepsilon: F \circ G \Rightarrow \text{id}_{\mathcal{D}}$

满足三角恒等式：
$$(\varepsilon F) \circ (F \eta) = \text{id}_F, \quad (G \varepsilon) \circ (\eta G) = \text{id}_G$$

```
┌─────────────────────────────────────────────────────────────────┐
│                      伴随函子结构                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│           C        F          D                                │
│           ● ════════════════→ ●                                │
│           │   ←────────────   │                                │
│           │        G          │                                │
│           ↓                   ↓                                │
│                                                                 │
│    伴随同构: Hom_D(F(X), Y) ≅ Hom_C(X, G(Y))                   │
│                                                                 │
│    单位 η: id_C → G∘F    余单位 ε: F∘G → id_D                   │
│                                                                 │
│         X ─────η_X────→ G(F(X))                               │
│                          │                                     │
│                          │                                     │
│         F(X) ←──ε_Y──── F(G(Y))                               │
│                                                                 │
│    三角恒等式:                                                 │
│                                                                 │
│         F ──Fη──→ FGF ──εF──→ F   =   id_F                     │
│         G ──ηG──→ GFG ──Gε──→ G   =   id_G                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**例 1.4.2（自由-遗忘伴随）**。遗忘函子 $U: \text{Grp} \to \text{Set}$ 的左伴随是**自由群函子** $F: \text{Set} \to \text{Grp}$，将集合 $S$ 映射到以 $S$ 为生成元的自由群 $F(S)$。伴随关系表现为：群同态 $F(S) \to G$ 一一对应于集合映射 $S \to U(G)$。

**例 1.4.3（像与原像）**。设 $f: X \to Y$ 为集合间的映射。则：
- 原像函子 $f^{-1}: \mathcal{P}(Y) \to \mathcal{P}(X)$ 是遗忘函子的右伴随
- 像函子 $f_*: \mathcal{P}(X) \to \mathcal{P}(Y)$ 是 $f^{-1}$ 的左伴随
- 存在伴随链：$f_! \dashv f^{-1} \dashv f_*$

**定理 1.4.4（伴随的唯一性）**。左伴随（若存在）在同构意义下唯一。即若 $F \dashv G$ 且 $F' \dashv G$，则 $F \cong F'$。

## 第二节：层作为层化结构的数学抽象——层范畴、层上同调与导出函子

### 2.1 层的几何起源与层公理

层（sheaf）论诞生于代数几何，但其思想根植于拓扑学与微分几何：如何系统地研究在局部有定义、且满足相容性条件的数学对象。

**定义 2.1.1（预层）**。设 $X$ 为拓扑空间，$\mathcal{C}$ 为范畴（通常为 $\text{Set}$, $\text{Grp}$, $\text{Ab}$, $\text{Ring}$）。$X$ 上取值于 $\mathcal{C}$ 的**预层** $\mathcal{F}$ 是反变函子：
$$\mathcal{F}: \text{Open}(X)^{\text{op}} \longrightarrow \mathcal{C}$$
其中 $\text{Open}(X)$ 是 $X$ 的开集范畴，以包含映射为态射。

具体而言，预层包含：
1. 对每个开集 $U \subseteq X$，指定对象 $\mathcal{F}(U) \in \text{Ob}(\mathcal{C})$（称为 $U$ 上的"截面"）；
2. 对包含映射 $V \subseteq U$，指定**限制映射** $\rho_{UV}: \mathcal{F}(U) \to \mathcal{F}(V)$。

满足：$\rho_{UU} = \text{id}$，且对 $W \subseteq V \subseteq U$，有 $\rho_{UW} = \rho_{VW} \circ \rho_{UV}$。

**定义 2.1.2（层）**。预层 $\mathcal{F}$ 称为**层**，如果满足以下**层公理**（sheaf axioms）：

设 $\{U_i\}_{i \in I}$ 为开集 $U$ 的开覆盖，$s_i \in \mathcal{F}(U_i)$ 为一族截面。

1. **局部性（Locality）**：若对所有 $i, j$，有 $\rho_{U_i, U_i \cap U_j}(s_i) = \rho_{U_j, U_i \cap U_j}(s_j)$，则存在唯一 $s \in \mathcal{F}(U)$ 使得 $\rho_{U, U_i}(s) = s_i$；
2. **粘合性（Gluing）**：对相容的局部截面族 $\{s_i\}$（即在交集上一致），存在整体截面 $s$ 限制到各 $U_i$ 上等于 $s_i$。

### 2.2 层范畴与态射

**定义 2.2.1（层态射）**。设 $\mathcal{F}, \mathcal{G}$ 为 $X$ 上的层。一个**层态射** $\phi: \mathcal{F} \to \mathcal{G}$ 是一族自然变换，即对每个开集 $U \subseteq X$，指定态射 $\phi_U: \mathcal{F}(U) \to \mathcal{G}(U)$，使得下图交换：
$$
\begin{array}{ccc}
\mathcal{F}(U) & \xrightarrow{\phi_U} & \mathcal{G}(U) \\
\rho_{UV} \downarrow & & \downarrow \rho'_{UV} \\
\mathcal{F}(V) & \xrightarrow{\phi_V} & \mathcal{G}(V)
\end{array}
$$

**定理 2.2.2（层范畴）**。拓扑空间 $X$ 上的层（取值于阿贝尔群范畴 $\text{Ab}$）构成**阿贝尔范畴**（abelian category），记作 $\text{Sh}(X, \text{Ab})$ 或简写为 $\text{Sh}(X)$。

阿贝尔范畴的关键性质：
- 存在零对象（零层）；
- 任意两个层有积与上积（在此范畴中二者同构）；
- 每个态射有核与上核；
- 单态射是核，满态射是上核。

**定义 2.2.3（茎与层化）**。设 $\mathcal{F}$ 为 $X$ 上的预层，$x \in X$。$\mathcal{F}$ 在 $x$ 处的**茎**定义为正向极限：
$$\mathcal{F}_x := \varinjlim_{x \in U} \mathcal{F}(U)$$

茎的元素称为**芽**（germ），记 $s_x \in \mathcal{F}_x$ 为截面 $s$ 在 $x$ 处的芽。

预层到层的**层化**（sheafification）是左伴随于遗忘函子的左正合函子，将预层转化为"最接近的"层。

### 2.3 层上同调与导出函子

层上同调的产生源于整体截面函子的非正合性。

**定义 2.3.1（整体截面函子）**。设 $X$ 为拓扑空间，$\text{Sh}(X)$ 为层范畴。**整体截面函子**定义为：
$$\Gamma(X, -): \text{Sh}(X) \longrightarrow \text{Ab}, \quad \mathcal{F} \mapsto \Gamma(X, \mathcal{F}) = \mathcal{F}(X)$$

**命题 2.3.2**。$\Gamma(X, -)$ 是左正合函子，但一般不是正合的。即对层的短正合列：
$$0 \longrightarrow \mathcal{F}' \longrightarrow \mathcal{F} \longrightarrow \mathcal{F}'' \longrightarrow 0$$
诱导的序列：
$$0 \longrightarrow \Gamma(X, \mathcal{F}') \longrightarrow \Gamma(X, \mathcal{F}) \longrightarrow \Gamma(X, \mathcal{F}'')$$
在末尾可能不满。

**定义 2.3.3（导出函子）**。设 $F: \mathcal{A} \to \mathcal{B}$ 为阿贝尔范畴间的左正合函子。$F$ 的**右导出函子** $R^i F$ 定义为：对 $A \in \text{Ob}(\mathcal{A})$，取内射分解 $A \to I^\bullet$，则：
$$R^i F(A) := H^i(F(I^\bullet))$$

**定义 2.3.4（层上同调）**。设 $\mathcal{F}$ 为 $X$ 上的层。**层上同调群**定义为整体截面函子的右导出函子：
$$H^i(X, \mathcal{F}) := R^i \Gamma(X, \mathcal{F})$$

**定理 2.3.5（Grothendieck 消失定理）**。设 $X$ 为拓扑维数 $n$ 的空间（如 $n$ 维流形），则对所有 $i > n$ 和任意层 $\mathcal{F}$，有 $H^i(X, \mathcal{F}) = 0$。

**定理 2.3.6（de Rham 定理）**。设 $M$ 为光滑流形，$\mathbb{R}$ 为常数层。则存在同构：
$$H^i_{\text{dR}}(M) \cong H^i(M, \mathbb{R})$$
其中左边为 de Rham 上同调，右边为层上同调。这建立了微分形式与拓扑不变量间的深刻联系。

```
│                                                                 │
│    0 → F → I^0 → I^1 → I^2 → I^3 → ...    (内射分解)            │
│         ↓    ↓    ↓    ↓                                         │
```

### 2.4 层的层化结构与几何实现

层论提供了一个普适框架，用于描述具有局部-整体相容性的数学结构。

**例 2.4.1（结构层）**。设 $X$ 为代数簇（或概形）。**结构层** $\mathcal{O}_X$ 定义为：
$$\mathcal{O}_X(U) = \{U \text{ 上的正则函数}\}$$
这是代数几何的核心对象，将几何空间与交换代数紧密联系。

**例 2.4.2（光滑函数的层）**。设 $M$ 为光滑流形。$C^\infty_M$ 为光滑函数层：
$$C^\infty_M(U) = C^\infty(U) = \{U \text{ 上的光滑实值函数}\}$$

**例 2.4.3（向量丛的截面层）**。设 $E \to M$ 为光滑向量丛。截面层 $\Gamma_E$ 定义为：
$$\Gamma_E(U) = \Gamma(U, E) = \{U \text{ 上的光滑截面}\}$$
层论视角将向量丛转化为局部自由层（locally free sheaf）。

**定理 2.4.4（Serre-Swan）**。设 $M$ 为紧 Hausdorff 空间。则 $M$ 上有限秩向量丛范畴等价于有限生成投射 $C(M)$-模范畴：
$$\text{Vect}(M) \cong \text{fg-proj-Mod}_{C(M)}$$
其中 $C(M)$ 为 $M$ 上的连续函数环。这表明层论/模论可以完全刻画拓扑向量丛。

### 2.5 导出范畴与三角结构

**定义 2.5.1（复形范畴）**。设 $\mathcal{A}$ 为阿贝尔范畴。**复形范畴** $C(\mathcal{A})$ 的对象为上链复形 $A^\bullet = (A^n, d^n: A^n \to A^{n+1})_{n \in \mathbb{Z}}$，满足 $d^{n+1} \circ d^n = 0$。

**定义 2.5.2（同伦范畴与导出范畴）**。复形同伦 $f \sim g$ 定义为存在同伦映射 $h: A^\bullet \to B^{\bullet-1}$ 使得 $f - g = d_B \circ h + h \circ d_A$。
- **同伦范畴** $K(\mathcal{A})$：对象是复形，态射是同伦类；
- **导出范畴** $D(\mathcal{A})$：对 $K(\mathcal{A})$ 关于拟同构（quasi-isomorphism）局部化得到。

**定理 2.5.3（导出范畴的泛性质）**。导出范畴 $D(\mathcal{A})$ 是使所有拟同构成为同构的"最小"范畴，且保持导出函子的自然定义。

导出范畴中的** distinguished triangles**（特异三角形）替代了短正合列：
$$A^\bullet \longrightarrow B^\bullet \longrightarrow C^\bullet \longrightarrow A^\bullet[1]$$
这构成了**三角范畴**（triangulated category）的核心结构。

## 第三节：张量范畴与量子对称性——编织范畴、模张量范畴与TQFT

### 3.1 张量范畴的基本结构

张量范畴（tensor category），又称幺半范畴（monoidal category），为量子对称性提供了统一的代数框架。

**定义 3.1.1（幺半范畴）**。**幺半范畴** $(\mathcal{C}, \otimes, \mathbf{1}, \alpha, \lambda, \rho)$ 包含：
1. 范畴 $\mathcal{C}$；
2. **张量积**函子 $\otimes: \mathcal{C} \times \mathcal{C} \to \mathcal{C}$；
3. **单位对象** $\mathbf{1} \in \text{Ob}(\mathcal{C})$；
4. **结合约束**（associator）$\alpha_{X,Y,Z}: (X \otimes Y) \otimes Z \xrightarrow{\sim} X \otimes (Y \otimes Z)$；
5. **左右单位约束** $\lambda_X: \mathbf{1} \otimes X \xrightarrow{\sim} X$, $\rho_X: X \otimes \mathbf{1} \xrightarrow{\sim} X$。

满足**五边形公理**和**三角形公理**。

```
┌─────────────────────────────────────────────────────────────────┐
│                   幺半范畴的相干公理                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    五边形公理 (Mac Lane 相干定理)                                │
│                                                                 │
│              ((X⊗Y)⊗Z)⊗W                                       │
│                 │    α_X,Y,Z⊗id_W                              │
│                 ↓                                               │
│    (X⊗(Y⊗Z))⊗W ────────→ X⊗((Y⊗Z)⊗W)                          │
│        │                    │                                   │
│    α   │                    │  id_X⊗α_Y,Z,W                   │
│        ↓                    ↓                                   │
│    X⊗((Y⊗Z)⊗W) ────────→ X⊗(Y⊗(Z⊗W))                          │
│                 α_X,Y⊗Z,W                                       │
│                                                                 │
│    三角形公理                                                    │
│                                                                 │

│         ρ_X⊗id_Y          id_X⊗λ_Y                             │

│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**定理 3.1.2（Mac Lane 相干定理）**。在幺半范畴中，由结合约束和单位约束生成的所有合理图表都交换。因此，可安全地省略括号，写作 $X \otimes Y \otimes Z$。

### 3.2 编织范畴与辫子群

**定义 3.2.1（编织范畴）**。**编织范畴**（braided category）是配备自然同构（**编织**）：
$$c_{X,Y}: X \otimes Y \xrightarrow{\sim} Y \otimes X$$
的幺半范畴，满足**六边形公理**：

$$
\begin{array}{ccc}
(X \otimes Y) \otimes Z & \xrightarrow{c_{X,Y} \otimes \text{id}} & (Y \otimes X) \otimes Z \\
\alpha \downarrow & & \downarrow \alpha \\
X \otimes (Y \otimes Z) & & Y \otimes (X \otimes Z) \\
\text{id} \otimes c_{X,Z} \downarrow & & \downarrow c_{Y, X \otimes Z} \\
X \otimes (Z \otimes Y) & \xleftarrow{\alpha^{-1}} & (Y \otimes Z) \otimes X
\end{array}
$$

以及对称的另一个六边形。

编织 $c_{X,Y}$ 可视化为拓扑编织：

```
┌─────────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────────┤

│                                                                 │
│        X      Y              Y      X                           │
│         \    /                \    /                            │
│          \  /                  \  /                             │
│           \/                    \/                              │
│           /\                    /\                              │
│          /  \                  /  \                             │
│         /    \                /    \                            │
│        Y      X              X      Y                           │
│                                                                 │
```

**例 3.2.2（向量空间的编织）**。有限维向量空间范畴 $\text{Vect}(k)$ 是编织范畴，编织为：
$$c_{V,W}(v \otimes w) = w \otimes v$$
此为**对称编织**（$c_{W,V} \circ c_{V,W} = \text{id}_{V \otimes W}$）。

**例 3.2.3（量子群表示）**。量子群 $U_q(\mathfrak{g})$ 的表示范畴是编织范畴，但编织非对称，由通用 $R$-矩阵定义：
$$c_{V,W} = \tau \circ R$$
其中 $\tau$ 为交换映射，$R$ 满足 Yang-Baxter 方程。这给出量子对称性的代数实现。

### 3.3 模张量范畴与拓扑序

**定义 3.3.1（刚性范畴与对偶）**。幺半范畴 $\mathcal{C}$ 称为**刚性**（rigid）的，如果每个对象 $X$ 有**左对偶** $X^*$ 和**右对偶** ${}^*X$，配备求值（evaluation）和余求值（coevaluation）态射：
$$\text{ev}_X: X^* \otimes X \to \mathbf{1}, \quad \text{coev}_X: \mathbf{1} \to X \otimes X^*$$
满足 Zig-Zag 恒等式。

**定义 3.3.2（模张量范畴）**。特征零域 $k$ 上的**模张量范畴**（modular tensor category, MTC）是满足以下条件的有限半单刚性编织范畴：

1. **编织**：$c_{X,Y}: X \otimes Y \to Y \otimes X$；
2. **扭转**（twist）：$
	heta_X: X \to X$，满足 $
	heta_{X \otimes Y} = c_{Y,X} \circ c_{X,Y} \circ (\theta_X \otimes \theta_Y)$；
3. **模条件**：S-矩阵 $S_{ij} = \text{tr}(c_{X_i, X_j^*} \circ c_{X_j^*, X_i})$ 非退化（可逆）。

**定理 3.3.3（Verlinde 公式）**。在模张量范畴中，融合规则系数可由 S-矩阵对角化：
$$N_{ij}^k = \sum_m \frac{S_{im} S_{jm} S_{km}^*}{S_{0m}}$$

这建立了表示论与拓扑不变量的深刻联系。

**例 3.3.4（共形场论的模张量范畴）**。有理共形场论（RCFT）的初态空间构成模张量范畴。例如：
- $SU(2)_k$ WZW 模型给出 Fibonacci 范畴（$k=1$）和 Ising 范畴（$k=2$）；
- 极小模型对应 Virasoro 代数的幺正最高权表示范畴。

### 3.4 拓扑量子场论的范畴表述

**定义 3.4.1（Atiyah-Segal 公理）**。$n$ 维**拓扑量子场论**（TQFT）是从 $n$ 维配边（cobordism）范畴 $\text{Cob}(n)$ 到向量空间范畴 $\text{Vect}(k)$ 的对称幺半函子：
$$Z: \text{Cob}(n) \longrightarrow \text{Vect}(k)$$

配边范畴的对象为 $(n-1)$ 维闭流形，态射为 $n$ 维配边（带边界的流形，边界为态射的定义域与值域的不交并）。

**定理 3.4.2（2D TQFT 的分类）**。2维 TQFT 等价于交换 Frobenius 代数：
$$\{\text{2D TQFT}\} \cong \{\text{交换 Frobenius 代数}\}$$

**定理 3.4.3（Reshetikhin-Turaev 构造）**。给定模张量范畴 $\mathcal{C}$，存在 3D TQFT 的显式构造：
$$Z_{\mathcal{C}}: \text{Cob}(3) \longrightarrow \text{Vect}(k)$$
将闭 3-流形 $M$ 映射到其 Witten-Reshetikhin-Turaev 不变量。

**定理 3.4.4（Turaev-Viro 不变量）**。给定球状融合范畴 $\mathcal{C}$（ fusion category），定义 3-流形 $M$ 的 Turaev-Viro 不变量：
$$TV_{\mathcal{C}}(M) = \sum_{\text{colorings}} \prod_v \text{weight}(v) \prod_e \text{dim}(e)$$
这等价于 3D TQFT 的分割函数。

### 3.5 范畴化与量子群

**定义 3.5.1（高阶范畴化）**。范畴化（categorification）是将集合/数提升为范畴/向量空间的过程。经典例子：
- 自然数 $n$ 范畴化为有限集范畴；
-  Euler 示性数范畴化为同调群；
- 量子不变量范畴化为 TQFT。

**定理 3.5.2（Khovanov 同调）**。Jones 多项式的范畴化——Khovanov 同调 $Kh^{i,j}(L)$ 满足：
$$\sum_{i,j} (-1)^i q^j \dim Kh^{i,j}(L) = V_L(q)$$
其中 $V_L(q)$ 为 Jones 多项式。这是纽结不变量范畴化的里程碑。

**定理 3.5.3（量子群范畴化）**。Lusztig 和 Khovanov-Lauda 独立实现了量子群 $U_q(\mathfrak{g})$ 的范畴化：
$$\dot{\mathcal{U}}_q(\mathfrak{g}) \text{ (2-范畴)} \xrightarrow{\text{split}} U_q(\mathfrak{g})$$
这提供了量子群表示的 diagrammatic 计算。

## 第四节：高阶范畴与高阶层化——∞-范畴、同伦类型论与高阶几何

### 4.1 高阶范畴的动机与定义

经典范畴论中，态射间的等价关系过于刚性。高阶范畴引入"态射之间的态射"，使得同伦信息得以保留。

**定义 4.1.1（严格 n-范畴）**。**严格 n-范畴**是范畴的 n 层迭代：
- 0-态射 = 对象
- 1-态射 = 通常的态射
- 2-态射 = 1-态射间的态射（"态射的态射"）
- ...
- n-态射 = (n-1)-态射间的态射

严格 n-范畴的组合律满足严格的结合性和交换性。

**定义 4.1.2（弱 n-范畴）**。**弱 n-范畴**（或 $(\infty, n)$-范畴）放松严格公理，允许同伦层次的交换性：
- 结合性在同伦意义下成立（由更高阶的约束条件保证）；
- Mac Lane 相干定理推广为"所有合理的高维图表交换"。

**定义 4.1.3（∞-范畴，Quasi-category）**。Joyal 和 Lurie 的模型：

**拟范畴**（quasi-category）是满足**内 Kan 条件**的单纯集 $X: \Delta^{\text{op}} \to \text{Set}$：
对任意 $0 < k < n$，映射 $X_n \to \Lambda^n_k$（第 $k$ 号 horn 的填充）是满射。

### 4.2 同伦类型论与统一基础

**定义 4.2.1（Martin-Löf 类型论）**。**同伦类型论**（Homotopy Type Theory, HoTT）是 Martin-Löf 类型论与 $
\infty$-群胚（弱 ∞-范畴，仅含可逆态射）的结合。

核心洞察：
- 类型 = 空间
- 项 = 点
- 等同类型 $a =_A b$ = 路径空间
- 路径间的等同 = 高阶路径（同伦）

**公理 4.2.2（单值性公理，Univalence Axiom）**。**单值性公理**（Voevodsky）表述为：
$$(A =_\mathcal{U} B) \simeq (A \simeq B)$$
即，类型的等同等价于等价（同伦等价）。这替代了集合论中的外延公理，使"同构即相等"。

**定理 4.2.3（高阶归纳类型）**。**高阶归纳类型**（Higher Inductive Types, HITs）允许通过生成元定义类型的点、路径和高阶路径。

例如，圆 $S^1$ 定义为：
- 基点：$\text{base}: S^1$
- 环路：$\text{loop}: \text{base} =_{S^1} \text{base}$

这直接编码了拓扑空间的胞腔结构。

### 4.3 导出代数几何与叠

**定义 4.3.1（导出环）**。**导出环**是带有额外同伦结构的环谱。形式上，导出环是 $E_\infty$-环谱（交换环谱）。

**定义 4.3.2（导出概形）**。**导出概形**（derived scheme）是环化空间 $(X, \mathcal{O}_X)$，其中：
- $X$ 为拓扑空间；
- $\mathcal{O}_X$ 为取值于导出环的层；
- 满足局部条件：局部同构于谱 $\text{Spec}(A)$，$A$ 为导出环。

**定义 4.3.3（叠，Stack）**。**叠**是满足下降条件的范畴纤维化 $F: \mathcal{X} \to \text{Sch}$。直觉上，叠是"带有自同构族的空间"。

**例 4.3.4（模叠）**。设 $g \geq 2$。**模叠** $\mathcal{M}_g$ 定义为：
$$\mathcal{M}_g(S) = \{S \text{ 上的亏格 } g \text{ 光滑曲线}/S\} / \cong$$
这是 Deligne-Mumford 叠，编码了代数曲线的模空间。

**定理 4.3.5（Lurie 的导出代数几何）**。存在 ∞-范畴的叠理论，其中：
- **导出叠**（derived stack）：$\mathcal{X}: \text{dRing}^{\text{op}} \to \text{Spaces}$；
- 包含经典叠作为截断对象。

这为相交理论（virtual fundamental classes）和枚举几何提供了自然框架。

### 4.4 高阶层化与滤过范畴

**定义 4.4.1（滤过范畴）**。范畴 $\mathcal{C}$ 配备**滤过** $\{F^p \mathcal{C}\}_{p \in \mathbb{Z}}$，满足：
$$\cdots \subseteq F^{p+1}\mathcal{C} \subseteq F^p\mathcal{C} \subseteq F^{p-1}\mathcal{C} \subseteq \cdots \subseteq \mathcal{C}$$
这是层化结构在范畴层次的体现。

**定义 4.4.2（t-结构）**。**t-结构**是导出范畴 $D(\mathcal{A})$ 的剖分：
$$D^{\leq 0} = \{X: H^i(X) = 0, \forall i > 0\}$$
$$D^{\geq 0} = \{X: H^i(X) = 0, \forall i < 0\}$$

**定理 4.4.3（BBD 分解定理）**。对紧簇的适当映射 $f: X \to Y$，导出直像可分解为简单 perverse 层的直和：
$$Rf_* \mathbb{Q}_X[n] \cong \bigoplus_i \mathcal{IC}_{\bar{Y}_i}(\mathcal{L}_i)$$
这是 Hodge 理论的高阶推广。

### 5.2 量子场论的 functorial 描述

**定义 5.2.1（Segal 公理）**。**共形场论**（CFT）是 2D 共形配边范畴到向量空间范畴的幺半函子：
$$Z: \text{ConfCob} \longrightarrow \text{Vect}$$

**定义 5.2.2（扩展 TQFT）**。遵循 **cobordism hypothesis**（Baez-Dolan，Lurie 证明）：

**定理 5.2.3（Cobordism Hypothesis）**。设 $\mathcal{C}$ 为具有对偶的对称幺松范畴。则 n 维扩展 TQFT 由 $\mathcal{C}$ 中的**完全对偶对象**（fully dualizable object）唯一确定：
$$\{\text{n-dim ExtTQFT to } \mathcal{C}\} \cong \{\text{Fully dualizable objects in } \mathcal{C}\}$$

这为 TQFT 的分类提供了完整的数学框架。

### 5.3 层化结构与物理实在

**命题 5.3.1（经典场作为层）**。经典场论中的场构型构成空间上的层：
$$\mathcal{F}(U) = \{\text{区域 } U \text{ 上的光滑场构型}\}$$
场的局部性原理正是层公理的物理体现。

**命题 5.3.2（量子场的层化）**。在代数量子场论（AQFT）中，可观测量赋值构成**余层**（cosheaf）：
$$\mathcal{A}: \text{Open}(M) \longrightarrow \text{C}^*\text{-Alg}, \quad U \mapsto \mathcal{A}(U)$$
满足等时局域性和缠结结构。

**定理 5.3.3（范畴化重整化）**。重整化群流可视为范畴的变形：
$$\text{RG}: \text{QFT}_\Lambda \longrightarrow \text{QFT}_{\Lambda'}$$
其中 $\text{QFT}_\Lambda$ 为截断 $\Lambda$ 处的量子场论范畴。Wilsonian 重整化的分阶段积分对应函子复合。

### 5.4 弦论与导出几何

**命题 5.4.1（B-膜作为导出对象）**。在 B-模型拓扑弦论中，B-膜对应 Calabi-Yau 流形 $X$ 上的导出范畴对象：
$$D^b(\text{Coh}(X))$$
膜间的开弦态射给出 Ext 群。

**定理 5.4.2（同调镜像对称）**。（Kontsevich 猜想，Givental、SYZ 等证明的部分情形）：
$$D^b\text{Fuk}(Y, \omega) \cong D^b(\text{Coh}(X))$$
辛流形 $(Y, \omega)$ 的 Fukaya 范畴等价于镜像 Calabi-Yau $X$ 的凝聚层导出范畴。

这是范畴论在弦论中的最深刻应用，将辛几何与复代数几何通过导出范畴桥接。

## 结论与展望

范畴论与层化结构为万物理论提供了前所未有的数学框架。从本章的系统性论述中，我们可以提炼出以下核心洞见：

**第一**，层化是物理实在的 organizational principle。从经典场的局域性到量子纠缠的非局域关联，再到拓扑序的 emergent 现象，层化结构（layered structure）以不同形式贯穿始终。

**第二**，范畴论提供了描述对称性的统一语言。从群到张量范畴，再到模张量范畴和 ∞-范畴，对称性的"层级提升"对应着物理理论的深化：
- 经典对称性 → 群作用
- 量子对称性 → 张量范畴表示
- 拓扑对称性 → MTC 中的编织结构
- 高阶对称性 → ∞-群作用

**第三**，导出结构与层上同调是连接几何与代数的桥梁。de Rham 定理、Riemann-Roch 定理、指标定理等深刻结果，都可在导出范畴框架内得到统一理解。

**第四**，物理定律的范畴化不仅是数学重构，更暗示着本体论的深层结构。弦图的图形演算、T QFT 的 functorial 描述、同调镜像对称的导出等价——这些或许暗示着，范畴结构是物理世界更为原初的存在形式。

**未来方向**：
1. **导出代数几何与量子场论**：将 Lurie 的导出叠理论应用于量子场论的 UV/IR 结构；
2. **同伦类型论的物理诠释**：探索单值性公理与规范对称性的深层联系；
3. **范畴化重整化群**：建立 Wilsonian 重整化的严格范畴框架；
4. **高阶 TQFT 的分类**：利用 Lurie 的 cobordism hypothesis 探索 4D TQFT 的完整分类。

正如 Grothendieck 所预见：数学不是研究对象本身，而是研究对象之间的关系——而范畴论正是这种关系之学的最高形式。在通向万物理论的征途上，范畴论与层化结构将继续作为我们最强大的概念工具，揭示物理世界深层的数学美学。
