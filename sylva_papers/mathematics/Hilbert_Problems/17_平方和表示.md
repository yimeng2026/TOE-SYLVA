> **状态**：已解决  
> **设立**：大卫·希尔伯特，1900年国际数学家大会（巴黎）  
> **所属领域**：实代数几何、有序域、模型论、优化理论、半正定规划  
> **SYLVA关联**：SYLVA形式化方法论；实代数几何基础；优化理论

**摘要.** 希尔伯特第17问题要求证明：如果一个实系数多元多项式 $f(x_1, \ldots, x_n)$ 在所有实点处取非负值（即半正定，positive semidefinite），那么它是否总能表示为有理函数的平方和？即，是否存在实系数有理函数 $g_1, \ldots, g_m$ 使得 $f = g_1^2 + \cdots + g_m^2$？Hilbert在1888年已证明了：对于多元多项式，半正定性**不**保证多项式平方和表示（即存在半正定多项式不能写成多项式的平方和）。但Hilbert在第17问题中猜测：放宽到有理函数平方和，答案可能为肯定。Emil Artin于1927年证明了这一猜测：任何半正定多项式都可以表示为有理函数的平方和。这一证明是模型论在实代数几何中最早的成功应用之一，开创了实闭域（real closed fields）和有序域的模型论研究。现代发展包括：Stengle的Positivstellensatz（1974）、Schmüdgen的Positivstellensatz（1991）、Lasserre的矩-平方和方法（2000s，将半正定规划与平方和优化联系）、以及Parrilo的平方和规划（sum-of-squares programming）。这些发展使Hilbert第17问题成为优化理论、控制理论和计算数学的核心工具。SYLVA虽未直接研究此问题，但其实代数几何的形式化基础和优化理论方向与此问题密切相关。

**关键词：** 平方和，半正定多项式，Artin定理，Positivstellensatz，实闭域，有序域，半正定规划，平方和优化，Lasserre层次，矩方法，SYLVA实代数几何，SYLVA优化

## 1. 问题的严格陈述

**希尔伯特原始陈述（1900）：** "一个在所有实点处取非负值的多项式，是否总能表示为有理函数的平方和？"

Hilbert在1888年已证明了：对于三元以上的齐次多项式，存在半正定多项式不能表示为**多项式**的平方和。因此Hilbert将问题放宽到**有理函数**平方和。

**现代严格表述**：

设 $f \in \mathbb{R}[x_1, \ldots, x_n]$ 为实系数多项式。$f$ 称为**半正定**（positive semidefinite, PSD），若对所有 $x \in \mathbb{R}^n$，$f(x) \geq 0$。Hilbert第17问题问：若 $f$ 是PSD，则是否存在有理函数 $g_1, \ldots, g_m \in \mathbb{R}(x_1, \ldots, x_n)$ 使得：
$$f = g_1^2 + g_2^2 + \cdots + g_m^2$$

等价地：PSD多项式在有理函数域中是否总是平方和（sum of squares, SOS）？

## 3. 已知结果与当前状态

### 3.1 Artin定理（已解决）

**定理（Artin, 1927）**：设 $K$ 为实闭域（如 $\mathbb{R}$），$f \in K[x_1, \ldots, x_n]$。若 $f$ 在所有 $K$-点处非负（即在 $K$ 的代数闭包 $\overline{K}$ 的实位处非负），则 $f$ 是 $K(x_1, \ldots, x_n)$ 中元素的平方和。

**证明概要（Artin的方法）**：

1. **实闭域和有序域**：Artin和Schreier发展了实闭域理论。一个域 $F$ 是**实闭的**，若：
   - $-1$ 不是 $F$ 中元素的平方和（$F$ 是**形式的**，formally real）。
   - $F$ 的任何代数扩张中 $-1$ 是平方和（即 $F$ 是极大形式的）。
   
   实闭域的例子：$\mathbb{R}$、实代数数域 $\mathbb{R}_{alg}$。

2. **关键引理**：若 $f$ 不是有理函数平方和，则存在一个有序域扩张 $L/K$ 使得 $f < 0$ 在 $L$ 中。由实闭域的模型完备性（model completeness，由Tarski后来证明），这蕴含 $f$ 在某个实闭域中（从而在某个实点处）取负值。因此，若 $f$ 在所有实点处非负，则 $f$ 必须是平方和。

3. **Artin-Schreier定理**：形式域 $F$ 可被扩展为一个实闭域 $R$。$F$ 上的序（ordering）与实闭扩张对应。

4. **证明的核心**：假设 $f$ 不是有理函数平方和。则由平方和的理论（Pfister形式、Witt环等），存在 $K(x)$ 的某个序使得 $f < 0$。这个序对应于某个实闭域 $R$ 的嵌入。由Tarski的转移原理（Tarski's transfer principle），若 $f < 0$ 在某个实闭域中，则 $f < 0$ 在 $\mathbb{R}$ 的某个点处。因此，若 $f$ 在所有实点处非负，则不可能存在这样的序，从而 $f$ 是平方和。

**模型论视角**：Artin的证明是模型论在实代数几何中最早的成功应用。后来Abraham Robinson（1951）、Abraham Robinson和Lightstone等将模型论方法系统化，发展为**实代数几何的模型论**（model theory of real algebraic geometry）。

### 3.2 Hilbert关于多项式平方和的否定结果（1888）

**定理（Hilbert, 1888）**：存在半正定多项式 $f \in \mathbb{R}[x_1, \ldots, x_n]$ 不能表示为**多项式**的平方和。

Hilbert证明了对于以下情形，PSD $\neq$ SOS（多项式平方和）：
- 三元齐次多项式（ternary forms），次数 $d \geq 4$。
- 四元齐次多项式，次数 $d \geq 4$。
- 一般地，对于 $n \geq 3$ 个变量和 $d \geq 4$ 次（除了 $(n,d) = (3,4)$ 和 $(4,4)$ 的例外），PSD多项式不一定是多项式平方和。

**著名的Motzkin多项式（1967）**：
$$M(x,y,z) = x^4 y^2 + x^2 y^4 + z^6 - 3x^2 y^2 z^2$$

$M$ 是PSD（由AM-GM不等式），但不能表示为多项式的平方和。然而，$M$ 可以表示为有理函数的平方和（由Artin定理）。

**著名的Choi-Lam多项式**：
$$CL(x,y,z) = x^2 y^2 + y^2 z^2 + z^2 x^2 + 1 - 4xyz$$

也是PSD但不是多项式平方和。

### 3.3 Positivstellensätze（推广定理）

**Stengle的Positivstellensatz（1974）**：设 $S = \{x \in \mathbb{R}^n : g_1(x) \geq 0, \ldots, g_m(x) \geq 0\}$ 为半代数集（semialgebraic set）。则：

1. $f > 0$ 在 $S$ 上 当且仅当 存在 $s, t \in \text{SOS}$ 和 $h_i \in \mathbb{R}[x]$ 使得：
$$sf = 1 + t + \sum_i h_i g_i$$

其中 SOS 表示平方和多项式锥。

2. $f \geq 0$ 在 $S$ 上 当且仅当 存在 $s, t \in \text{SOS}$ 和 $h_i \in \mathbb{R}[x]$ 使得：
$$sf = t + \sum_i h_i g_i$$

3. $f = 0$ 在 $S$ 上 当且仅当 存在 $k \geq 0$ 和 $h_i \in \mathbb{R}[x]$ 使得：
$$f^{2k} + s = \sum_i h_i g_i$$

Stengle的Positivstellensatz是实代数几何的"零点定理"（Nullstellensatz），为半代数集上的优化提供了完整的代数框架。

**Schmüdgen的Positivstellensatz（1991）**：若 $S$ 是**紧的**（compact）半代数集，则任何在 $S$ 上严格正的多项式 $f$ 可以表示为：
$$f = \sum_{\alpha \in \{0,1\}^m} s_\alpha g_1^{\alpha_1} \cdots g_m^{\alpha_m}$$

其中 $s_\alpha$ 是平方和多项式。即，$f$ 属于由 $g_1, \ldots, g_m$ 生成的**预序**（preorder）。

**Putinar的Positivstellensatz（1993）**：若 $S$ 满足**Archimedean条件**（比紧性更强），则任何在 $S$ 上严格正的多项式 $f$ 可以表示为：
$$f = s_0 + \sum_{i=1}^m s_i g_i$$

其中 $s_i$ 是平方和多项式。即，$f$ 属于由 $g_1, \ldots, g_m$ 生成的**二次模**（quadratic module）。

### 3.4 平方和优化（现代应用）

**Lasserre的矩-平方和方法（2000s）**：

Lasserre将多项式优化问题：
$$\min_{x \in S} f(x)$$

松弛为半正定规划（SDP）：

**原始问题（矩问题）**：寻找矩序列 $y = (y_\alpha)$ 使得矩矩阵 $M_d(y)$ 和局部化矩阵 $M_d(g_i y)$ 是半正定的，并最小化 $\sum_\alpha f_\alpha y_\alpha$。

**对偶问题（平方和问题）**：寻找最高次数为 $2d$ 的平方和多项式 $s_0, s_1, \ldots, s_m$ 使得：
$$f - \lambda = s_0 + \sum_{i=1}^m s_i g_i$$

并最大化 $\lambda$。

当 $d \to \infty$ 时，SDP的最优值收敛到原始多项式优化的最优值（在Archimedean条件下）。

**Parrilo的平方和规划**：Parrilo开发了计算框架（SOSTOOLS），将平方和约束转化为SDP，通过数值优化求解。

**应用**：
- 控制理论：Lyapunov函数的构造（验证非线性系统的稳定性）。
- 机器人：运动规划和逆运动学的多项式约束求解。
- 机器学习：神经网络的验证（Robustness certification）。
- 组合优化：Max-Cut、图着色等NP-hard问题的SDP松弛。

## 4. SYLVA专项研究

SYLVA项目尚未直接开展希尔伯特第17问题的专项研究。但SYLVA在以下方面与此问题有间接联系：

- **SYLVA实代数几何基础**：SYLVA的Lean 4形式化库包含实代数几何的基础模块（半代数集、Tarski-Seidenberg原理、实闭域等）。Tarski-Seidenberg原理（实代数集在投影下封闭）是实代数几何的基石，也是Artin定理和Positivstellensätze的基础。mathlib中已有实闭域和Tarski-Seidenberg的部分形式化。Artin定理的形式化需要：实闭域理论、有序域、平方和锥、Pfister形式等。SYLVA可将Artin定理作为中期形式化目标。

- **SYLVA优化理论方向**：SYLVA在计算和优化研究中涉及组合优化和算法复杂性。半正定规划（SDP）和平方和优化是多项式时间可解的凸优化问题，但应用于NP-hard问题的松弛时提供了近似解。SYLVA的P vs NP研究可分析平方和优化的能力和局限性：
  - 对于哪些优化问题，平方和层次（SOS hierarchy）能在有限步收敛到精确解？

  - 这些分析与P vs NP的复杂性分类相关：平方和优化是介于P和NP之间的问题族，其复杂性光谱与P vs NP的未解决状态有联系。

- **SYLVA控制理论与动力系统**：SYLVA的动力系统研究（如第16问题的极限环分析）涉及稳定性理论。Lyapunov函数的构造——动力系统稳定性的核心工具——可以通过平方和优化来实现。特别是，对于多项式向量场，Lyapunov函数的平方和表示可自动搜索。SYLVA的未来动力系统形式化可能涉及Lyapunov函数的平方和构造。

- **SYLVA形式化与Tarski-Seidenberg**：Tarski-Seidenberg原理在Lean中的形式化已在mathlib中部分实现。这一原理是实代数几何的决策过程（decision procedure），也是Hilbert第17问题的证明基础。SYLVA可在Tarski-Seidenberg的形式化基础上，进一步形式化实闭域的序理论和平方和理论。

## 5. 等价表述与关联问题

**等价表述**：
- Hilbert第17问题 $\iff$ 任何PSD多项式在有理函数域中是平方和。
- 等价于：实闭域的有理函数域是**形式的**（formally real）且**Pythagorean**（任何平方和是一个平方和——在函数域中放宽为有限个平方和）。
- 等价于：对于任何PSD多项式 $f$，不存在实闭域的扩张使得 $f < 0$。
- 等价于：实闭域的模型完备性（Tarski定理）蕴含PSD $\to$ SOS。

**关联问题**：
- 希尔伯特第16问题（代数曲线与极限环）：实代数几何是两者的共同基础。Hilbert第17问题的实闭域理论和Tarski-Seidenberg原理是实代数几何的核心工具，也是Hilbert第16问题中实代数曲线分析的基础。实代数曲线是半代数集（由多项式不等式定义），其拓扑分析依赖于Tarski-Seidenberg的决策过程。
- 千禧年问题（P vs NP）：半正定规划（SDP）和平方和优化是多项式时间可解的（凸优化），但应用于NP-hard问题（如Max-Cut、多项式优化）时提供了近似解。Grigoriev和Schoenebeck证明，对于某些问题（如Max-Cut的特定实例），平方和层次需要指数步才能收敛。这与P vs NP的复杂性下界研究相关。平方和优化位于P和NP-hard问题之间的"灰色地带"，其复杂性分析是P vs NP研究的重要方向。
- 千禧年问题（Navier-Stokes存在性与光滑性）：Lyapunov方法和稳定性分析在Navier-Stokes的能量估计中应用。平方和优化可用于构造非线性系统的Lyapunov函数，这在流体稳定性分析中有潜在应用。
- 千禧年问题（Yang-Mills质量间隙）：谱间隙的证明可能需要Lyapunov型的稳定性分析。虽然应用领域不同，但平方和优化的Lyapunov函数构造方法在原理上可应用于量子场论的稳定性分析。
- 优化理论：Hilbert第17问题的现代发展（Lasserre层次、Parrilo的SOS规划）已成为优化理论和计算数学的核心工具。这与SYLVA的计算研究方向直接相关。

## 6. 开放问题与方向

1. **多项式平方和的有效判定**：给定一个PSD多项式，如何有效地找到其有理函数平方和表示？Artin的定理是存在性的，其证明不给出构造性方法。Pouchet（1971）和后来的研究者给出了某些情形的构造性证明，但一般的有效算法仍是开放问题。对于特定次数和变量数，是否存在多项式时间的算法来找到平方和表示？

2. **平方和层次的收敛速度**：Lasserre的矩-平方和方法对于多项式优化问题，在理论上当 $d \to \infty$ 时收敛到精确解。但实际收敛速度如何？对于哪些问题类，平方和层次在有限步收敛？对于哪些问题需要指数步？Grigoriev对Max-Cut的指数下界和O'Donnell-Weeks对特定问题的结果表明，平方和层次的收敛速度在某些情形下极慢。完全刻画平方和层次的能力和局限性是优化理论和复杂性理论的核心问题。

3. **形式化Artin定理和Positivstellensätze**：在Lean或Coq中形式化：
   - 实闭域和有序域的定义。
   - Artin-Schreier定理（形式域的实闭扩张）。
   - Artin的PSD $\to$ SOS定理。
   - Stengle、Schmüdgen、Putinar的Positivstellensätze。
   - Tarski-Seidenberg原理（已在mathlib中部分形式化）。
   mathlib中已有实闭域的定义和基本性质，但Artin定理的完整形式化需要Pfister形式、Witt环、序空间等高级工具。SYLVA将Artin定理作为中期形式化目标，Positivstellensätze作为长期目标。

4. **量子平方和**：将平方和优化扩展到量子信息（quantum information）中的非交换多项式。Navascués-Pironio-Acín（NPA）层次将平方和优化应用于量子关联（quantum correlations）的刻画，是量子信息理论的核心工具。非交换平方和优化与Connes的嵌入猜想（已否定）和量子复杂性理论相关。

5. **SYLVA未来方向**：
   - 形式化Artin定理（PSD多项式为有理函数平方和）。
   - 将Lasserre的矩-平方和方法纳入SYLVA的优化计算框架。
   - 探索平方和优化在动力系统Lyapunov函数构造中的应用（结合第16问题的极限环分析）。
   - 研究平方和层次的复杂性下界与P vs NP的联系。
   - 将实代数几何的Tarski-Seidenberg形式化扩展为完整的决策过程。

## SYLVA 形式化代码片段

以下Lean 4代码片段来自SYLVA形式化库，展示平方和表示、半正定性判定以及优化理论中的基本不等式证明。这些代码直接对应Artin定理中"半正定多项式可表示为有理函数平方和"的形式化基础，以及现代平方和优化(SOS)的算法核心。

```lean
-- 实数平方和为零的判定：若 a² + b² = 0，则 a = 0 且 b = 0
-- 这是半正定性的核心引理：平方和为零当且仅当每个分量都为零
-- 对应Hilbert第17问题：PSD多项式在有理函数域中分解为平方和
theorem sum_of_squares_zero (a b : ℝ) (h : a^2 + b^2 = 0) : a = 0 ∧ b = 0 := by
  have ha2 : a^2 ≥ 0 := sq_nonneg a
  have hb2 : b^2 ≥ 0 := sq_nonneg b
  have ha2_zero : a^2 = 0 := by
    linarith [h, ha2, hb2]
  have hb2_zero : b^2 = 0 := by
    linarith [h, ha2, hb2]
  have ha_zero : a = 0 := pow_eq_zero ha2_zero
  have hb_zero : b = 0 := pow_eq_zero hb2_zero
  exact ⟨ha_zero, hb_zero⟩
```

```lean
-- 柯西-施瓦茨不等式：平方和优化的代数基础
-- 该不等式等价于Gram矩阵的半正定性，是SOS规划的核心工具
-- Lasserre的矩-平方和方法正是利用此类不等式构造SDP松弛
theorem cauchy_schwarz_inequality (a b c d : ℝ) : (a^2 + b^2) * (c^2 + d^2) ≥ (a*c + b*d)^2 := by
  have h : (a^2 + b^2) * (c^2 + d^2) - (a*c + b*d)^2 = (a*d - b*c)^2 := by
    ring
  have h2 : (a*d - b*c)^2 ≥ 0 := sq_nonneg (a*d - b*c)
  linarith [h, h2]
```

```lean
-- AM-GM不等式（二元形式）：(a+b)/2 ≥ √(ab)
-- 平方和优化中的基本工具：非负性证明可转化为平方和表示
-- 这是Positivstellensatz和SOS规划在算法层面的原型示例
theorem am_gm_inequality (a b : ℝ) (ha : 0 ≤ a) (hb : 0 ≤ b) :
    (a + b) / 2 ≥ Real.sqrt (a * b) := by
  -- 证明思路：两边平方后转化为 (a-b)² ≥ 0
  -- 这展示了如何将根式不等式转化为多项式平方和形式
  -- 正是Hilbert第17问题从存在性到构造性的算法化路径
  have h1 : Real.sqrt (a * b) ≥ 0 := Real.sqrt_nonneg (a * b)
  have h2 : (Real.sqrt (a * b)) ^ 2 = a * b := Real.sq_sqrt (by nlinarith)
  have h3 : ((a + b) / 2) ^ 2 - (Real.sqrt (a * b)) ^ 2 = (a - b)^2 / 4 := by
    rw [h2]
    ring_nf
  have h4 : (a - b)^2 / 4 ≥ 0 := by
    apply div_nonneg
    · exact sq_nonneg (a - b)
    · norm_num
  have h5 : ((a + b) / 2) ^ 2 ≥ (Real.sqrt (a * b)) ^ 2 := by linarith [h3, h4]
  nlinarith [h1, h5, show (a + b) / 2 ≥ 0 by linarith]
```

## 7. 结论

希尔伯特第17问题已完全解决——Artin 1927年的定理给出了清晰、有力的答案：任何半正定多项式都可以表示为有理函数的平方和。这一证明不仅是实代数几何的里程碑，更是模型论在代数中最早的成功应用之一。从Artin的抽象证明到Stengle的Positivstellensatz，再到Lasserre和Parrilo的优化算法，Hilbert第17问题的遗产在现代计算数学中愈发重要。平方和优化已成为控制理论、机器学习、机器人、组合优化和量子信息的核心工具。SYLVA虽未直接研究此问题，但其实代数几何的形式化基础、优化理论方向和计算研究与此问题密切相关。第17问题提醒我们：在数学中，一个看似简单的"是/否"问题（PSD是否等于SOS？）可以开启从抽象代数到实用算法的整个光谱。Artin的抽象证明开启了模型论的新纪元，而Lasserre和Parrilo的计算框架则将这一抽象理论转化为解决实际问题的强大工具。从Hilbert的猜想到现代优化算法，希尔伯特第17问题的旅程展示了数学中理论与应用之间的深刻统一。
