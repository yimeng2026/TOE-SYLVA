> **状态**：已解决  
> **设立**：大卫·希尔伯特，1900年国际数学家大会（巴黎）  
> **所属领域**：代数数论、二次型、局部-整体原理、算术几何  
> **SYLVA关联**：SYLVA形式化方法论（一般数学知识应用）；数论基础；Local-Global模板

---

**摘要.** 希尔伯特第11问题要求解决代数数域上的二次型理论。具体而言，给定系数在代数数域 $K$ 中的二次型 $Q(x_1, \ldots, x_n)$，问：是否存在有理数域上的算法来判定该方程 $Q(x) = 0$ 是否有非平凡解？更一般地，问：对于代数数域 $K$ 上的任意二次型，是否存在非平凡的 $K$-有理点？Hilbert本人在《数论报告》（Zahlbericht, 1897）中已部分解决了实数域上的二次型理论（惯性定律）。第11问题的完全解决由Helmut Hasse（1923–1924）的局部-整体原理（Hasse-Minkowski定理）给出：二次型在全局域 $K$ 上有非平凡解当且仅当它在 $K$ 的所有完备化（局部域 $K_v$）上都有非平凡解。Hasse-Minkowski定理是局部-整体原理的典范，也是现代代数数论和算术几何的基石。此外，对于正定二次型，高斯（Gauss）的二元二次型理论、Minkowski的数的几何、Siegel和Tate的 $\theta$-函数方法等，为二次型的算术理论提供了丰富的工具。Hilbert第11问题已完全解决。SYLVA虽未直接研究此问题，但其Local-Global模板设计与Hasse-Minkowski的局部-整体原理深刻共鸣。

**关键词：** 二次型，Hasse-Minkowski定理，局部-整体原理，代数数域，Witt定理，高斯二元型，数的几何，SYLVA数论，Local-Global模板

## 1. 问题的严格陈述

**希尔伯特原始陈述（1900）：** "关于系数在代数数域中的二次型：给定系数在代数数域 $K$ 中的二次型 $Q(x_1, \ldots, x_n)$，是否存在非平凡的 $K$-有理点？即方程 $Q(x) = 0$ 是否有非零解 $x \in K^n$？"

Hilbert还进一步问：是否存在有理数域上的算法来判定这一问题？以及，对于不定二次型，是否存在有理整数点？

**现代严格表述**：

设 $K$ 为代数数域（或更一般地，全局域——数域或函数域）。设 $Q$ 为 $K$ 上的非退化二次型（即 $Q \in K[x_1, \ldots, x_n]$ 是二次齐次多项式，且其判别式非零）。Hilbert第11问题问：

1. **存在性**：$Q(x) = 0$ 是否有非平凡 $K$-解？即是否存在 $x \in K^n \setminus \{0\}$ 使得 $Q(x) = 0$？
2. **算法性**：是否存在算法来判定上述问题？
3. **整数解**：对于 $K = \mathbb{Q}$，$Q(x) = 0$ 是否有非平凡整数解？

## 3. 已知结果与当前状态

### 3.1 Hasse-Minkowski定理（已解决）

**定理（Hasse-Minkowski, 1923–1924）**：设 $K$ 为全局域（数域或函数域），$Q$ 为 $K$ 上的非退化二次型（$n \geq 2$）。则 $Q$ 在 $K$ 上有非平凡零点（即 $Q(x) = 0$ 有非零 $K$-解）当且仅当 $Q$ 在 $K$ 的所有完备化 $K_v$（$v$ 遍历 $K$ 的所有位）上都有非平凡零点。

**证明概要**：

1. **局部-整体原理**：Hasse-Minkowski定理是**局部-整体原理**（local-global principle）在二次型上的典范实例。局部-整体原理断言：对于某些类型的"全局"问题，其在全局域上的可解性等价于在所有局部域上的可解性。

2. **$\mathbb{Q}$ 的情形**：对于 $K = \mathbb{Q}$，$\mathbb{Q}$ 的完备化包括：
   - 实完备化：$\mathbb{Q}_\infty = \mathbb{R}$
   - $p$-adic 完备化：$\mathbb{Q}_p$（对每个素数 $p$）

   在 $\mathbb{R}$ 上，二次型 $Q$ 有非平凡解当且仅当 $Q$ 是**不定的**（indefinite，即既有正特征值也有负特征值）。
   在 $\mathbb{Q}_p$ 上，二次型的可解性由Hilbert符号的乘积公式（Hilbert第9问题）决定：对于三元二次型，$Q$ 在 $\mathbb{Q}_p$ 上可解当且仅当某些不变量满足条件。

3. **证明技术**：Hasse的证明使用Hilbert符号和乘积公式。对于二元二次型，问题归结为范数方程（norm equation），由局部类域论解决。对于高维二次型，使用Witt分解和归纳法。

4. **算法性**：Hasse-Minkowski定理不仅给出了存在性判据，还给出了**算法**。判定 $Q$ 在 $\mathbb{Q}$ 上是否有非平凡解：
   - 检查 $Q$ 在 $\mathbb{R}$ 上是否不定（计算惯性指数，需要 $O(n^3)$ 运算）。
   - 检查 $Q$ 在每个 $\mathbb{Q}_p$ 上是否可解。由强逼近定理，只需检查有限个 $p$（判别式的素因子）。
   - 因此，这是一个**可判定的**问题。

**定理（Witt, 1937）**：设 $Q$ 为域 $F$（特征 $\neq 2$）上的非退化二次型。任何两个等距的二次型空间具有相同的Witt分解（Witt decomposition）：$Q \cong Q_{\text{hyp}} \perp Q_{\text{anis}}$，其中 $Q_{\text{hyp}}$ 是双曲平面（hyperbolic planes）的直和，$Q_{\text{anis}}$ 是各向异性的（anisotropic，即无非平凡零点）。

### 3.2 整数解与表示问题

对于 $K = \mathbb{Q}$，Hilbert还关心**整数解**（$x \in \mathbb{Z}^n$）的存在性。这一问题比 $K$-有理解更复杂。

**定理（Siegel, 1935）**：对于正定二次型 $Q$，$Q$ 在 $\mathbb{Z}$ 上表示给定整数 $m$ 的方式数（representation number）可通过模形式（modular forms）和 $\theta$-函数来计算。具体而言，$Q$ 的 $\theta$-级数定义为：
$$\Theta_Q(\tau) = \sum_{x \in \mathbb{Z}^n} e^{\pi i \tau Q(x)}$$

这是模形式（对于某些同余子群），其Fourier系数给出表示数。Siegel证明了表示数的精确公式（Siegel公式），涉及局部密度和Tamagawa数。

**定理（Meyer's theorem）**：任何 $\mathbb{Q}$ 上的不定二次型，若变量数 $n \geq 5$，则自动有非平凡整数零点。对于 $n = 4$，需要额外的条件（由局部-整体原理和强逼近定理处理）。

### 3.3 高维推广与代数群

Hasse-Minkowski定理的推广到代数群上：

**猜想（Hasse原理 for algebraic groups）**：对于连通线性代数群 $G$ 在全局域 $K$ 上的齐性空间 $X = G/H$，$X(K) \neq \emptyset$ 当且仅当 $X(K_v) \neq \emptyset$ 对所有位 $v$。

这一原理对**二次型**（正交群 $O(Q)$ 的齐性空间）成立（Hasse-Minkowski），但对更一般的代数群**不成立**。这引出了**Galois上同调**和**Tamagawa数**的研究。

**Tamagawa数**：对于代数群 $G$，Tamagawa数 $\tau(G)$ 定义为adele群上的测度。Weil猜想：对于单连通半单群，$\tau(G) = 1$。这已被证明（Ono, Weil, Kottwitz等），并隐含了Hasse原理的某些形式。

## 4. SYLVA专项研究

SYLVA项目尚未直接开展希尔伯特第11问题的专项研究。但SYLVA在以下方面与此问题有间接联系：

- **SYLVA Local-Global模板**：SYLVA的Local-Global模板设计（提取Cook-Levin、BSD、Hodge、Riemann问题的共同Local-to-Global结构）与Hasse-Minkowski的局部-整体原理深刻共鸣。Hasse-Minkowski定理是局部-整体原理的典范实例：
  - 局部信息（每个完备化 $K_v$ 上的可解性）决定全局信息（$K$ 上的可解性）。
  - SYLVA的Local-Global模板分析其他问题（如BSD猜想：局部L-因子的乘积决定整体L-函数；Hodge猜想：局部上同调的整体性质；Cook-Levin定理：局部布尔满足性决定全局可满足性）。
  - 在Hasse-Minkowski中，局部-整体原理**成立**（对于二次型）。但在其他问题（如Hilbert第10问题的某些变体、Pell方程的广义版本）中，局部-整体原理**不成立**。SYLVA的模板分析这种成立与不成立的条件，寻找深层的结构规律。

- **SYLVA数论形式化基础**：SYLVA的Lean 4形式化库包含有限域、代数数域、Galois理论等基础模块。二次型的形式化（包括Witt分解、Hasse-Minkowski定理）在理论上是可行的。mathlib中已有二次型（quadratic forms）的模块定义，包括Witt消去和Clifford代数。Hasse-Minkowski定理的完整形式化需要$p$-adic数、Hilbert符号、强逼近定理等高级工具。SYLVA可将此作为中期形式化目标。

- **与BSD猜想的联系**：椭圆曲线的二次扭（quadratic twists）和BSD猜想。对于椭圆曲线 $E: y^2 = x^3 + ax + b$，其二次扭 $E_d: dy^2 = x^3 + ax + b$ 的L-函数与原始曲线的L-函数通过二次特征相联系。这涉及局部-整体原理在椭圆曲线上的应用（BSD猜想是局部-整体原理在算术几何中的核心实例）。

- **与P vs NP的联系**：二次型的整数解问题（对于不定二次型）在 $n \geq 5$ 时是自动可解的（Meyer's theorem），因此是平凡的。对于 $n = 3, 4$，判定整数解的存在性需要更复杂的分析。这与P vs NP中的某些问题（如二次约束的满足性）有联系：二次型的整数解在特定条件下是NP-hard的，但在一般条件下可由局部-整体原理高效判定。

## 5. 等价表述与关联问题

**等价表述**：
- Hilbert第11问题（$K$-有理解）$\iff$ 由Hasse-Minkowski定理，局部可解性等价于全局可解性。
- Hilbert第11问题（算法性）$\iff$ 可判定：通过检查有限个局部域（由判别式的素因子）和实域的惯性指数，可在多项式时间内判定。
- 对于整数解：不定二次型的整数解存在性在 $n \geq 5$ 时自动成立；$n = 3, 4$ 时由局部-整体原理和强逼近定理处理。

**关联问题**：
- 希尔伯特第9问题（一般互反律）：Hilbert符号和Hilbert互反律（乘积公式）是Hasse-Minkowski定理证明的核心工具。Hilbert符号 $(a, b)_v$ 在二次型的判别式和可解性中起关键作用。Hasse-Minkowski定理与类域论（Hilbert第9问题）密切相连。
- 希尔伯特第12问题（Kronecker-Weber推广）：二次型的代数数域理论与类域论相关。二元二次型的类群与理想类群相关。
- 千禧年问题（BSD猜想）：椭圆曲线的二次扭与BSD猜想。局部-整体原理在椭圆曲线上以BSD猜想的形式出现。Hasse-Minkowski定理是BSD猜想的"二次型类比"。
- 千禧年问题（P vs NP）：二次约束的满足性（QP）在特定形式下是NP-hard的。局部-整体原理的有效计算与复杂性分类。
- 希尔伯特第10问题（丢番图方程）：二次Diophantine方程（如Pell方程 $x^2 - Dy^2 = 1$）是Hilbert第10问题中少数可判定的情形之一。Hasse-Minkowski定理和连分数理论给出了Pell方程的完整解。
- 希尔伯特第17问题（平方和表示）：Artin的解决使用实闭域和半正定的二次型表示。Hilbert第17问题与二次型的正定性密切相关。

## 6. 开放问题与方向

1. **高次型的Hasse原理**：对于三次型、四次型等，Hasse-Minkowski的局部-整体原理**不成立**。经典反例：Selmer的三次型 $3x^3 + 4y^3 + 5z^3 = 0$ 在局部有解但无全局解。判定高次型是否有全局解是开放问题（Hilbert第10问题的一部分）。

2. **有效Hasse-Minkowski**：Hasse-Minkowski定理给出了可解性的判据，但找到**具体的解**（而非仅仅存在性）的算法效率问题。对于 $\mathbb{Q}$ 上的三元二次型，找到具体的有理解涉及模 $p$ 的解提升（Hensel引理）和实解的连续性。有效算法和计算复杂性是活跃方向。

3. **形式化Hasse-Minkowski**：在Lean或Coq中形式化Hasse-Minkowski定理。这需要：
   - 二次型理论（Witt分解、Clifford代数）。
   - $p$-adic数和局部域。
   - Hilbert符号和互反律。
   - 强逼近定理。
   - 局部可解性的判定算法。
   mathlib中已有部分模块（二次型、$p$-adic数），但完整的Hasse-Minkowski形式化仍是全球开放目标。SYLVA将此作为中期形式化目标。

4. **Tamagawa数和代数群**：Weil的Tamagawa数猜想（单连通半单群的Tamagawa数为1）已被证明，但一般代数群的Tamagawa数计算仍是活跃问题。Tamagawa数与L-函数的特殊值（BSD公式中的Sha群和调节子）相关。

5. **SYLVA未来方向**：
   - 将Hasse-Minkowski定理的陈述和局部-整体原理形式化于Lean 4。
   - 探索Local-Global模板在二次型、椭圆曲线和BSD猜想中的统一结构。
   - 研究二次型整数解的算法与P vs NP中二次约束满足性问题的联系。
   - 将Hilbert符号和Hilbert互反律（Hilbert第9问题）与二次型理论的形式化整合。

## SYLVA 形式化代码片段

以下Lean 4代码片段来自SYLVA形式化库，展示局部-整体原理的核心类型结构以及二次型相关的不等式证明技巧。这些代码体现了第11问题中Hasse-Minkowski定理背后的形式化基础：局部数据与全局解之间的结构化对应。

```lean
-- 局部问题定义：每个Hilbert问题在局部层面都有一个对应的结构
-- 对应二次型的局部可解性：在每个完备化 K_v 上判定是否有解
structure LocalProblem (L : Type*) where
  -- 局部相容性条件：何时局部数据可以"粘合"
  compatibility : L → Prop
  -- 局部问题可判定性实例
  decidableCompat : ∀ l, Decidable (compatibility l)

-- 全局问题定义：局部问题通过"下降"(descent)操作得到全局结果
-- 对应Hasse-Minkowski：局部可解性 → 全局可解性
structure GlobalProblem (G : Type*) where
  -- 全局解的存在性条件
  solutionExists : G → Prop
  -- 全局可判定性实例
  decidableSol : ∀ g, Decidable (solutionExists g)
```

```lean
-- 柯西-施瓦茨不等式的代数形式：与二次型的正定性密切相关
-- 定理：(a²+b²)(c²+d²) ≥ (ac+bd)²，等号成立当且仅当 (a,b) 与 (c,d) 线性相关
-- 这正是二次型 Q(x,y) = x² + y² 的半正定性的直接推论
theorem cauchy_schwarz_2D (a b c d : ℝ) : (a^2 + b^2) * (c^2 + d^2) ≥ (a*c + b*d)^2 := by
  have h : (a^2 + b^2) * (c^2 + d^2) - (a*c + b*d)^2 = (a*d - b*c)^2 := by
    ring
  have h2 : (a*d - b*c)^2 ≥ 0 := sq_nonneg (a*d - b*c)
  linarith [h, h2]
```

```lean
-- 实数平方和为零的判定：若 a² + b² = 0，则 a = 0 且 b = 0
-- 这是二次型 Q(x,y) = x² + y² 正定性（事实上是正定）的核心性质
-- 与Hasse-Minkowski定理中"不定二次型在局部有解"形成对比：
-- 正(半)定二次型在实局部域 R 上可解性由惯性指数完全决定
theorem sum_of_squares_zero (a b : ℝ) (h : a^2 + b^2 = 0) : a = 0 ∧ b = 0 := by
  have ha2 : a^2 ≥ 0 := sq_nonneg a
  have hb2 : b^2 ≥ 0 := sq_nonneg b
  have ha2_zero : a^2 = 0 := by
    linarith [h, ha2, hb2]
  have hb2_zero : b^2 = 0 := by
    linarith [h, ha2, hb2]
  have ha_zero : a = 0 := pow_eq_zero ha2_zero
  have hb_zero : b = 0 := pow_eq_zero hb2_zero
  exact ⟨ha_zero, hb_zero⟩
```

## 7. 结论

希尔伯特第11问题已完全解决——Hasse-Minkowski定理给出了清晰、有力的答案：二次型在全局域上的可解性等价于在所有局部域上的可解性。这一定理不仅是Hilbert第11问题的答案，更是局部-整体原理的典范，是现代代数数论、算术几何和代数群理论的基石。Witt的分解定理、Siegel的表示公式和Tate的adelic语言，使二次型的算术理论成为数学中最精密的分支之一。然而，Hasse-Minkowski定理的局部-整体原理在高次型上不成立，这揭示了二次型作为"临界"结构的特殊性。SYLVA虽未直接研究此问题，但其Local-Global模板与Hasse-Minkowski的局部-整体原理深刻共鸣，其形式化能力和数论基础为未来深入这一经典领域提供了准备。第11问题提醒我们：在数学中，局部与整体之间的关系是深刻的——有时局部决定整体（如二次型），有时局部不决定整体（如三次型），而理解这种决定与不决定的条件，正是数学的核心追求之一。
