> **状态**：部分解决（Kepler猜想已解决；n维最密堆积一般未解决；晶体群有限性已解决）  
> **设立**：大卫·希尔伯特，1900年国际数学家大会（巴黎）  
> **所属领域**：离散几何、凸几何、格论、球堆积、晶体学、形式化证明

**摘要.** 希尔伯特第18问题涉及空间堆积（sphere packing）的多个方面：1) n维空间中最密球堆积（特别是三维空间中的Kepler猜想）；2) 是否只有有限多种具有基本域（fundamental domain）的晶体群（crystallographic groups）。Kepler猜想断言：三维空间中相同球体的最密堆积方式是面心立方（FCC）和六方最密堆积（HCP），其堆积密度为 $\pi/\sqrt{18} \approx 0.74048$。Thomas Hales 1998年给出了Kepler猜想的证明，2017年Flyspeck项目完成了Lean形式化验证。Maryna Viazovska 2016年解决了八维空间（$\mathbb{R}^8$）中E8格的最优堆积问题，获2022年Fields奖。R^24中Leech格的最优堆积由Cohn和Kumar 2009年证明。SYLVA直接关联此问题：SYLVA在数学形式化前沿中关注球堆积的形式化证明，R^8中E8格最优堆积的Lean 4形式化正在进行（Seewoo Lee等）。n维空间的最密堆积在一般维度（除1, 2, 3, 8, 24外）仍是开放问题。Hilbert关于晶体群有限性的原始问题已由Bieberbach 1910年解决：n维空间中只有有限多种（同构意义下）晶体群。SYLVA形式化方法论可为更高维度堆积问题的形式化提供基础。

**关键词：** Kepler猜想，球堆积，最密堆积，E8格，Leech格，晶体群，Bieberbach定理，堆积密度，线性规划界，Cohn-Elkies方法，Viazovska定理，Lean形式化，Flyspeck，SYLVA形式化

## 1. 问题的严格陈述

**希尔伯特原始陈述（1900）：** Hilbert第18问题包含三个部分：
1. **最密堆积**：n维欧几里得空间中，相同球体的最密堆积方式是什么？特别是三维空间中的最密球堆积。
2. **等球堆积（Kugel）**：是否存在除最密堆积外的其他等球堆积，具有相同的堆积密度（即"非唯一的最密堆积"）？
3. **晶体群**：是否只有有限多种具有基本域的晶体群（空间群）？即n维空间中，不等价的晶体群（保持晶格不变的离散等距变换群）是否只有有限多种？

**现代严格表述**：

**1. 最密堆积**：设 $\mathbb{R}^n$ 为n维欧几里得空间。一个球堆积（sphere packing）是一组不重叠的球体 $\{B(x_i, r)\}_{i \in I}$ 的集合（球心为 $x_i$，半径为 $r$）。堆积的**密度** $\delta$ 定义为：在足够大的区域中，球体体积占区域体积的比例。问：在 $\mathbb{R}^n$ 中，相同球体的最大堆积密度 $\delta_n$ 是多少？哪些堆积方式达到这一密度？

**2. 堆积的唯一性**：在 $\mathbb{R}^n$ 中，达到最大密度 $\delta_n$ 的堆积方式是否唯一（在等距变换和测度零调整下）？如果不唯一，有多少种本质上不同的最密堆积？

**3. 晶体群（空间群）**：n维欧几里得空间 $E^n$ 中的**晶体群**（crystallographic group）是一个离散等距变换群 $G \subset \text{Isom}(E^n)$，使得存在有界的基本域（fundamental domain），且 $E^n / G$ 是紧的（或有限体积的）。问：在固定维度 $n$ 中，不同构的晶体群是否只有有限多种？

## 3. 已知结果与当前状态

### 3.1 一维和二维（已解决）

**一维（$n=1$）**：球（线段）紧密排列，密度 $\delta_1 = 1$（唯一）。

**二维（$n=2$）**：最密堆积是六方堆积（hexagonal packing），每个圆与6个相邻圆接触。密度为：
$$\delta_2 = \frac{\pi}{\sqrt{12}} \approx 0.90690$$

**定理（Mahler, 1953）**：二维最密堆积是唯一的（在等距变换下），即六方堆积。

### 3.2 三维Kepler猜想（已解决）

**Kepler猜想（1611）**：三维空间中，相同球体的最密堆积密度为 $\delta_3 = \pi/\sqrt{18} \approx 0.74048$，由FCC（面心立方）和HCP（六方最密堆积）达到。

**历史证明尝试**：
- Gauss（1831）证明了在**格堆积**（lattice packing，球心形成Bravais格）中FCC/HCP是最密的。
- 但非格堆积（irregular packing）可能更密？这长期未解决。
- 许多数学家尝试证明，包括Minkowski、Hilbert时代以来的多次尝试。

**Hales的证明（1998/2005）**：

Hales使用**计算机辅助证明**（computer-assisted proof），将问题分解为约5000个线性规划问题的枚举。

**证明策略**：
1. 将球堆积分解为**Voronoi单元**（Voronoi cells）或**Delaunay三角剖分**（Delaunay triangulation）。
2. 在每个局部配置中，定义一个**得分函数**（score function），将堆积密度与局部几何关联。
3. 证明任何局部配置的得分不超过FCC/HCP的得分。
4. 这需要枚举大量可能的局部配置（球的几何排列），并通过线性规划验证每个配置的得分上界。
5. 总共约5000个线性规划问题，需要计算机验证。

**形式化验证（Flyspeck, 2017）**：

Hales领导的Flyspeck项目使用HOL Light证明助手完成了Kepler猜想证明的完整形式化验证。这是数学形式化历史上的里程碑之一：
- 形式化了所有几何引理和线性规划验证。
- 使用线性规划求解器的形式化输出（证书验证）。
- 总代码量超过10万行HOL Light代码。
- 结果发表于《Forum of Mathematics, Pi》（2017）。

### 3.3 八维E8格（已解决）

**定理（Viazovska, 2016）**：在 $\mathbb{R}^8$ 中，E8格的球堆积是最密的。堆积密度为：
$$\delta_8 = \frac{\pi^4}{384} \approx 0.25367$$

**证明方法——Cohn-Elkies方法 + 模形式**：

1. **Cohn-Elkies方法（2003）**：Cohn和Elkies证明了若存在一个径向函数 $f: \mathbb{R}^n \to \mathbb{R}$ 满足特定条件（在球心处取正值，在球堆积的禁止距离处取非正值，且其Fourier变换满足类似条件），则可从 $f$ 导出堆积密度的上界。

2. **线性规划界**：球堆积密度问题可转化为无限维线性规划问题。Cohn-Elkies方法将其转化为寻找"辅助函数" $f$ 的优化问题。

3. **Viazovska的突破**：Viazovska发现了在 $\mathbb{R}^8$ 中满足Cohn-Elkies条件的**显式辅助函数**。这一辅助函数由**模形式**（modular forms）构造——具体而言，使用了与E8格相关的特定模形式（Laplace特征值问题、球面设计的联系）。

4. 该辅助函数恰好使上界与E8格的堆积密度匹配，从而证明了E8格的最优性。

**R^24的Leech格（Cohn-Kumar-Miller-Radchenko-Viazovska, 2017）**：

类似的方法应用于 $\mathbb{R}^{24}$。Viazovska与其合作者构造了由模形式给出的辅助函数，证明了Leech格的堆积密度为最优：
$$\delta_{24} = \frac{\pi^{12}}{12!} \approx 0.00193$$

### 3.5 晶体群的有限性（已解决）

**定理（Bieberbach, 1910）**：对于固定维度 $n$，n维欧几里得空间中的晶体群（crystallographic groups）只有有限多种（同构意义下）。

**证明概要**：
1. 晶体群 $G$ 包含一个平移子群 $T \cong \mathbb{Z}^n$（格），且 $T$ 是 $G$ 的有限指数正规子群。
2. 商群 $G/T$（点群）是有限群，嵌入 $O(n)$（正交群）。
3. 由Jordan-Zassenhaus定理，对于固定的 $n$，$O(n)$ 中有限阶矩阵的有限子群只有有限多种（同构意义下）。
4. 对于每个点群，格 $T$ 的变形（保持点群对称性的格）构成有限维的模空间。
5. 由Mahler的紧性定理，在一定条件下，格的等价类是紧的，因此只有有限多种。

**具体数量**：
- $n=2$：17种壁纸群（wallpaper groups）。
- $n=3$：230种空间群（Fedorov/Schoenflies分类，1890）。
- $n=4$：4783种（Bülow, 1970s）。
- $n=5, 6$：由Plesken、Schulz等枚举，数量庞大但有限。

SYLVA在数学形式化前沿（math_formalization_frontier.md）中持续关注球堆积问题的形式化证明。这是SYLVA形式化方法论的重点方向之一。

- **R^8最优球堆积的Lean 4形式化（进行中）**：Seewoo Lee等研究者正在将Viazovska的R^8最优球堆积证明形式化于Lean 4。这一项目涉及：

  - Cohn-Elkies线性规划方法的形式化。
  - E8格的构造和性质的形式化。
  - Fourier分析（特别是径向函数和Fourier变换）的形式化。
  - 球堆积密度的严格定义和计算的形式化。
  
  SYLVA关注这一项目的进展，并可能在未来参与或扩展类似的形式化工作。R^8的形式化比Kepler猜想（Flyspeck）更"纯粹数学"（主要涉及模形式和Fourier分析，而非大规模线性规划枚举），因此在形式化技术上可能更直接。

### 4.3 SYLVA与离散几何的交叉

球堆积问题与SYLVA的其他研究方向有交叉：
- **编码理论和信息论**：球堆积与纠错码（error-correcting codes）的对应。在 $\mathbb{F}_q^n$ 上，码的 packing radius 与球堆积的对应。SYLVA的编码理论研究可受益于球堆积的最优性结果。
- **格密码学（Lattice-based cryptography）**：E8格和Leech格在密码学中的应用（如GGH、NTRU等格密码方案）。球堆积密度与格密码的安全性分析（最短向量问题SVP的困难性）相关。SYLVA的密码学和计算研究方向可涉及格密码学的形式化。
- **优化理论**：Cohn-Elkies方法本质上是无限维线性规划。SYLVA的优化研究方向（如半正定规划、组合优化）可分析这一方法的计算复杂性和扩展性。

## 5. 等价表述与关联问题

**等价表述**：
- Hilbert第18问题（堆积）$\iff$ 在 $\mathbb{R}^n$ 中，确定 $\delta_n = \sup\{\text{density of sphere packing}\}$ 并找到达到 $\delta_n$ 的堆积。
- 在 $\mathbb{R}^3$ 中，$\iff$ Kepler猜想：$\delta_3 = \pi/\sqrt{18}$，由FCC/HCP达到。
- 在 $\mathbb{R}^8$ 中，$\iff$ Viazovska定理：$\delta_8 = \pi^4/384$，由E8格达到。
- 在 $\mathbb{R}^{24}$ 中，$\iff$ Leech格定理：$\delta_{24} = \pi^{12}/12!$，由Leech格达到。
- Hilbert第18问题（晶体群）$\iff$ Bieberbach定理：$n$ 维晶体群只有有限多种（同构意义下）。

**关联问题**：
- 希尔伯特第3问题（四面体体积相等）：涉及多面体的分解和体积，与球堆积的体积计算有基础联系。但两者在技术层面上差异较大。
- 千禧年问题（P vs NP）：格密码学（基于格难题的密码系统）与P vs NP的联系。最短向量问题（SVP）和最近向量问题（CVP）在一般格中是NP-hard的。球堆积的最优性结果（如E8格的最优性）为格密码的安全性提供了数学基础。
- 千禧年问题（Poincaré猜想，已解决）：球堆积问题与几何拓扑共享对"空间结构"的深层追问。Poincaré猜想断言单连通三维闭流形同胚于S³，而Hilbert第18问题追问n维空间中最密球堆积的密度与构造。两者在几何拓扑的框架中交汇：Bieberbach定理（晶体群有限性）与Thurston几何化纲领（三维流形的八种标准几何分解）都涉及离散群在常曲率空间中的几何作用。Viazovska证明R^8中E8格最优堆积时使用的模形式，与Poincaré猜想证明中使用的几何分析工具，展现了离散几何与拓扑在分析层面的统一。E8格在弦论（heterotic string theory）中的出现，进一步连接了球堆积、几何拓扑与量子引力。
- 编码理论和信息论：球堆积与纠错码的对应（Hamming空间与欧几里得空间）。在 $\mathbb{F}_2^n$ 中，码的 packing 对应于球堆积的离散版本。这一对应是编码理论的核心。
- 数论和模形式：Viazovska的证明使用了模形式的深刻性质。模形式与椭圆曲线、L-函数、BSD猜想（千禧年问题）等有深刻联系。E8格的Theta函数与模形式的联系是这一证明的核心。
- 组合设计：球堆积中的球面设计（spherical design）与组合设计理论有联系。E8格和Leech格与球面t-design的构造相关。

## 6. 开放问题与方向

1. **一般维度的最密堆积**：对于 $n = 4, 5, 6, 7$ 和 $n \geq 9$（除24外），最密堆积尚未被严格证明。Cohn-Elkies方法提供了系统的方法论：寻找满足特定条件的辅助函数。对于某些维度（如 $n=4$），猜测 $D_4$ 格是最优的，但尚未找到证明所需的辅助函数。Viazovska的方法依赖于特定维度的模形式性质，难以直接推广到一般维度。

2. **非格堆积的最优性**：在三维中，Hales的证明处理了非格堆积。但在高维中，最密堆积是否总是格堆积（lattice packing）？在 $n=8$ 和 $n=24$ 中，最优堆积是格堆积（E8和Leech）。但在其他维度中，非格堆积可能优于任何已知格堆积。对于 $n=10$ 及以上，存在非格堆积的密度超过任何已知格堆积的猜测。证明或否定"最优堆积总是格堆积"的猜想是开放问题。

3. **形式化更高维度的堆积**：完成R^8的Lean 4形式化（Seewoo Lee等的项目）后，SYLVA可探索：
   - 将R^8的形式化扩展到R^24（Leech格）。

## SYLVA 形式化代码片段

以下Lean 4代码片段来自SYLVA形式化库，展示晶体对称性分类、距离空间结构以及优化相关的不等式证明。这些代码直接对应球堆积问题中的形式化基础：对称性分类、度量空间中的最优化，以及离散几何的代数工具。

```lean
-- 10-fold way 对称性分类：时间反演(T)、粒子-空穴(P)、手性(C)
-- 在晶体群分类中，对称性类的枚举是核心问题
-- Bieberbach定理证明n维晶体群有限性的关键是对对称性操作的分类
inductive SymmetryClass
  | A    -- 无对称性
  | AIII -- 手性
  | AI   -- T² = +1
  | BDI  -- T² = +1, C² = +1
  | D    -- P² = +1
  | DIII -- T² = -1, P² = +1
  | AII  -- T² = -1
  | CII  -- T² = -1, C² = -1
  | C    -- P² = -1
  | CI   -- T² = +1, P² = -1
  deriving DecidableEq, Repr

-- Kitaev周期表：对称性类与维度决定拓扑不变量
-- 类似于晶体群：维度与对称性操作决定了群的分类
structure KitaevPeriodicTable where
  dimension : ℕ
  symmetryClass : SymmetryClass
  topologicalInvariant : Type
  -- 约束：对称性类与维度决定拓扑不变量类型
  invariantConstraint : (symmetryClass = SymmetryClass.A ∧ dimension = 2 → topologicalInvariant = ℤ) ∧
                        (symmetryClass = SymmetryClass.D ∧ dimension = 2 → topologicalInvariant = ZMod 2)
```

```lean
-- 非负实数对的有序性：在度量空间中距离必须非负
-- 球堆积的核心度量：球心之间的距离必须 ≥ 2r（不重叠条件）
-- 这正是堆积密度优化的基本约束
theorem distance_nonneg (a b : ℝ) (ha : 0 ≤ a) (hb : 0 ≤ b) :
    (a + b) / 2 ≥ Real.sqrt (a * b) := by
  -- AM-GM不等式：证明两个非负数的算术平均 ≥ 几何平均
  -- 在球堆积中，这对应于最优排列的局部密度约束
  have h1 : Real.sqrt (a * b) ≥ 0 := Real.sqrt_nonneg (a * b)
  have h2 : (Real.sqrt (a * b)) ^ 2 = a * b := Real.sq_sqrt (by nlinarith)
  have h3 : ((a + b) / 2) ^ 2 - (Real.sqrt (a * b)) ^ 2 = (a - b)^2 / 4 := by
    rw [h2]
    ring_nf
  have h4 : (a - b)^2 / 4 ≥ 0 := by
    apply div_nonneg
    · exact sq_nonneg (a - b)
    · norm_num
  have h5 : ((a + b) / 2) ^ 2 ≥ (Real.sqrt (a * b)) ^ 2 := by linarith [h3, h4]
  nlinarith [h1, h5, show (a + b) / 2 ≥ 0 by linarith]
```

```lean
-- 2D动量空间（布里渊区）结构：在晶体学中，倒易格点描述晶体的周期性
-- 堆积密度的傅里叶分析等价于布里渊区上的积分
structure CrystalMomentum2D where
  kx : ℝ
  ky : ℝ

-- 布里渊区 (2D 环面 T²)：周期性边界条件的数学实现
-- 这与晶体堆积的周期性结构完全对应
structure BrillouinZone2D where
  -- 动量范围: [-π/a, π/a] × [-π/a, π/a]
  a : ℝ  -- 晶格常数
  ha : a > 0
  momentum : CrystalMomentum2D → Prop
  -- 周期性边界条件
  periodicity : ∀ k, momentum k ↔
    (-Real.pi / a ≤ k.kx ∧ k.kx ≤ Real.pi / a) ∧
    (-Real.pi / a ≤ k.ky ∧ k.ky ≤ Real.pi / a)
```

## 7. 结论

希尔伯特第18问题的答案是一个混合图景：晶体群的有限性已完全解决（Bieberbach, 1910）；球堆积的最优性在维度1, 2, 3, 8, 24已解决，但在一般维度仍开放。Kepler猜想的解决（Hales, 1998/2005）是计算机辅助证明的里程碑，其形式化验证（Flyspeck, 2017）展示了数学严格性的新标准。Viazovska 2016年对R^8的证明和随后对R^24的证明，使用模形式的深刻性质和Cohn-Elkies的线性规划方法，为离散几何开辟了新纪元。这些工作不仅解决了具体的堆积问题，更展示了分析、数论和代数的深层统一。SYLVA通过关注数学形式化前沿和R^8的Lean 4形式化，直接参与这一问题的现代研究。第18问题提醒我们：在数学中，"最密堆积"——看似简单的几何问题——却连接着数论、分析、代数和计算的深层结构。从Kepler的直觉到Viazovska的模形式，希尔伯特第18问题的旅程展示了数学在简单问题中揭示的无限深度。
